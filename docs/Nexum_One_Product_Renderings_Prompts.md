## 渲染风格指南

| 属性 | 值 |
|------|-----|
| 配色 | 哑光黑 (#1a1a1a)、深蓝 (#1e3a5f)、Nexum 蓝 (#0096f0)、银色 (#c0c0c0) |
| 材质感 | 软触感硅胶、阳极氧化铝、哑光碳纤维、高级运动纺织品 |
| 整体调性 | 医疗级可信度 + 消费科技优雅感 + 运动服的舒适感 |
| 避免 | 铬/亮面塑料、科幻机器人美学、笨重外骨骼框架、可见线缆 |
| 目标 | 产品应当看起来像你**想要**穿戴的东西，而不是你**不得不**穿戴的东西 |
