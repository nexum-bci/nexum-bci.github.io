## 1. 安全警告 (Safety Warnings)

### 1.1 电池安全 (Battery Handling)

**危险：** 6S Li-ion 21700 电池组（22.2 V 标称）储存能量约 50 Wh。错误操作可导致火灾、爆炸或化学灼伤。

- **绝缘防护：** 所有电池连接必须使用 Kapton 胶带或热缩管覆盖裸露触点。组装中任何金属工具（扳手、镊子）不得同时接触正负极。
- **BMS 先接：** 电池组必须先连接 BMS 再接入 PCB。BMS 未激活前，主电路保持断开。
- **充电限制：** 仅使用指定充电器（USB-C PD 20V/3A 或 6S Li-ion 平衡充）。充电时必须在防火袋中。
- **机械保护：** 电池组在壳体内必须用 2mm 硅胶垫或 EVA 泡沫完全包裹，防止振动力传递到电芯。
- **损坏检查：** 任何凹陷、鼓包或漏液的电芯立即停止使用，按当地法规处理。
- **存放：** 长期不用时存储电量为 40-60% (约 3.8V/电芯)，存放温度 10-25°C。

### 1.2 碳纤维粉尘 (Carbon Fiber Dust)

**警告：** 碳纤维切割/打磨产生的粉尘导电且刺激皮肤和呼吸道。

- **必须：** 戴口罩 (N95 或以上)、护目镜、长袖工作服。禁止裸手接触碳纤维切片。
- **环境：** 在通风良好或带吸尘的工作站操作。碳纤维粉尘会损坏电子设备（导电）。
- **处置：** 碳纤维废料密封袋装，标注"碳纤维废料 — 导电"。不可混入普通垃圾。
- **打磨：** 碳纤维件必须湿磨（浸水打磨）或使用 HEPA 吸尘器。

### 1.3 线缆张力危险 (Cable Tension Hazards)

**警告：** Bowden 线缆在工作时承受高达 800N 的张力。断裂的线缆如同鞭子：

- **测试时保持距离：** 首次通电张力测试时，所有人员距离线缆路径至少 1 米。
- **防护罩：** 绕线轮区域必须安装透明防护罩（原型阶段可使用亚克力板）。
- **机械熔断器：** 线缆末端必须串联 800N 机械熔断器（见 BOM 线缆终端套件）。
- **检查：** 每次使用前目视检查线缆有无断股、扭结或磨损。

### 1.4 一般安全

- **ESD：** 所有 PCB 操作必须在 ESD 垫上进行，佩戴 ESD 腕带。
- **高温：** 电机驱动器和电池区域在运行时可达 60°C。组装后需张贴高温警示标签。
- **锐边：** 6061 铝壳 CNC 后可能有锐边，需用倒角工具或砂纸(400#)处理所有边缘。
- **热风枪：** 使用热缩管时注意高温，远离易燃物。

## 3. 组装序列总览

```
步骤 1: 控制盒壳体准备
  |
步骤 2: 控制盒内部 PCB 堆叠 + 电池安装
  |
步骤 3: 电机总成 + 绕线轮组装
  |
步骤 4: Bowden 线缆裁剪 + 终端处理
  |
步骤 5: 碳纤维髋部锚点加工 + 预装配
  |
步骤 6: 压缩衣改装 —— 碳纤维锚点集成
  |
步骤 7: 压缩衣改装 —— sEMG 电极缝制
  |
步骤 8: Bowden 线缆布线 + 固定
  |
步骤 9: 总装 —— 电机-线缆-锚点连接
  |
步骤 10: 最终质量检查
```

**预计总工时 (第一台):** 16-24 小时（含固化时间）
**预计总工时 (第二台):** 10-14 小时（经验累积）

## 4. 分步组装说明

---

### Step 1: 控制盒壳体准备

**目标：** 准备好 6061 铝合金上下壳体，包括所有螺纹孔、密封槽和标识。

#### 1.1 来料检查
- 确认壳体 CNC 加工完成，表面无毛刺、无划痕、无加工残留。
- 使用游标卡尺检查关键尺寸：
  - 外形：120 +/-0.2 x 80 +/-0.2 x 30 +/-0.2 mm
  - 壁厚：2.0 +/-0.1 mm
  - 密封槽深度：1.0 +/-0.1 mm，宽度 2.2 +/-0.1 mm

#### 1.2 去毛刺
- 使用 400# 砂纸或倒角工具处理所有非配合边缘，特别注意：
  - USB-C 开口处（防止刮伤数据线插头）
  - 电机线缆出线孔
  - 散热鳍片（如有）
- 用 IPA 和无尘布清洁所有表面。

#### 1.3 螺纹孔检查
使用对应的螺丝进行螺纹测试。每孔应能用手指轻松旋入螺丝至少 3-4 圈。

| 螺纹尺寸 | 位置 | 数量 | 用途 |
|----------|------|------|------|
| M3 x 0.5 | 壳体四角 | 4 | 上下壳固定 |
| M3 x 0.5 | 内部支柱 x4 | 4 | PCB 固定 |
| M3 x 0.5 | 电池仓两侧 | 2 | 电池压板 |
| M4 x 0.7 | 壳体底部（外部） | 4 | 腰夹安装 |
| M3 x 0.5 | 侧面板 | 2 | 电机安装架固定 |

- 如发现螺纹不畅，使用丝锥过一遍。
- 在每一颗螺丝孔内滴一小滴 Loctite 243 备用。

#### 1.4 密封圈安装
- 将 phi2mm 硅胶 O 型圈放入下壳的密封槽中。
- O 型圈不可扭曲，两端对接处使用少量氰基丙烯酸酯胶水（瞬干胶）固定。
- 在 O 型圈上薄涂一层硅脂（方便以后拆卸）。

#### 1.5 导热垫贴合
- 在下壳底部内侧，对应 MOSFET/驱动器位置，贴 2mm 导热硅胶垫。
- 尺寸裁剪为 30x30mm（每块驱动器区域一块）。
- 使用滚轮压实，确保无气泡。

**验证点：** 壳体清洁，螺纹通畅，O 型圈平顺，导热垫贴合无气泡。

### Step 2: 控制盒内部 PCB 堆叠与电池安装

**目标：** 完成控制盒内部电子组件的机械安装。

```
                   +-------------------------+
                   |       上壳 (TOP)        |
                   | 6061 Aluminum, 2mm wall |
                   |                         |
                   |   +-----------------+   |
                   |   |   nRF5340 BLE   |   |
                   |   |   Module (子板) |   |
                   |   +--------+--------+   |
                   |            | UART       |
                   |   +--------+--------+   |
                   |   |   STM32H743     |   |
                   |   |   主控板 (PCB)  |   |
                   |   +--------+--------+   |
                   |            | FFC        |
                   |   +--------+--------+   |
                   |   |   DRV8316      |   |
                   |   |   电机驱动板   |   |
                   |   +--------+--------+   |
                   |            |             |
                   |   +--------+--------+   |
                   |   |  6S 21700      |   |
                   |   |  Battery Pack  |   |
                   |   |  EVA泡沫包裹   |   |
                   |   +-----------------+   |
                   +-------------------------+
                        |         |
                   导热垫 --> 下壳散热
                   (3W/mK silicone pad)
```

#### 2.1 主控板 (STM32H743) 安装
- 将主控板对准四角的 M3 支柱。
- 使用 M3 x 6mm 内六角螺钉 + M3 弹簧垫圈固定。
- 扭力：0.4 Nm。
- **不要接触 PCB 中心区域（BGA 底部），不要按压 STM32 芯片。**

#### 2.2 BLE 模块 (nRF5340) 安装
- nRF5340 作为子板，通过排针插座连接到主控板。
- 将排针对齐、压入，确保无弯曲。
- 如使用独立模块，通过 M2 螺丝固定于预留孔位（M2 螺丝扭矩 0.2 Nm）。

#### 2.3 电机驱动板 (DRV8316) 安装
- 驱动板通过 FFC/FPC 柔性排线连接至主控板。
- 将驱动板放置在导热垫上，确保芯片（DRV8316）与导热垫直接接触。
- 使用 M3 x 6mm 螺丝固定，扭矩 0.4 Nm。
- **关键：** 驱动板与导热垫之间可涂薄层导热硅脂以填充微小间隙，但不要过量（避免溢出导致短路）。

#### 2.4 IMU 传感器
- BMI270 IMU 模块（或预贴在主控板上的 IMU）已焊接在主控板上。

#### 2.5 电池组件安装

**电池组件预组装（在控制盒外完成）：**

```
   +------[BMS PCB]------+
   |   BQ76952 BMS PCB    |
   +--+--+--+--+--+--+
      |  |  |  |  |  |
     C1 C2 C3 C4 C5 C6
      |  |  |  |  |  |
   +--+--+--+--+--+--+
   |  6S 21700 Battery   |
   |  Pack (Samsung 50E) |
   |  6S1P               |
   +---------------------+
```

- 6S 21700 电芯已由电池厂点焊成组（自带镍片）。
- BMS (BQ76952) 按 BMS 接线图连接至每组电池。
  - B+ -> 电池正极
  - B1-B6 -> 各节电池间连接点
  - B- -> 电池负极
- 所有连接点用 Kapton 胶带覆盖绝缘。
- 整体用 EVA 泡沫板包裹（2mm 厚，双面胶固定）。
- **在包入 EVA 之前，用万用表确认：**
  - 总电压：22.2-25.2 V（取决于充电状态）
  - 各节单体电压差 < 0.05 V
  - B- 到 P- 不导通（BMS 未激活时）
- 将包裹好的电池组件放入壳体电池仓。
- 使用 M3 x 8mm + 压板固定，扭矩 0.4 Nm。
- 电池输出线（XT60 或 XT30 连接器）暂时不接主板，用胶带绝缘端子。

#### 2.6 接线整理
- 使用扎带整理电池线、电机线（穿过侧壁出线孔）。
- 确保所有线缆不挡住壳体密封边缘。
- 在下壳对应出线孔位置安装 PG7 或 PG9 电缆防水接头。

**验证点：** PCB 安装稳固，电池固定无晃动，导热垫贴合良好，所有电气连接双重绝缘。

### Step 3: 电机总成与绕线轮组装

**目标：** 将 T-motor AK80-9（或 AK60-6）装配绕线轮，准备线缆卷绕。

```
              +--------------+
              |  绕线轮 Spool|
              |  (铝合金 CNC)|
              |    phi40x20mm |
              +-------+------+
                      | M4 中心轴
              +-------+------+
              | T-motor      |
              | AK80-9 BLDC  |
              | 12V/24V      |
              | 输出轴 phi8mm |
              +--------------+
                      |
              +-------+------+
              | 电机安装架    |
              | 6061铝 CNC   |
              | 固定于控制盒  |
              +--------------+
```

#### 3.1 绕线轮加工（如从 BOM 采购 CNC 件则跳过）

如 3D 打印绕线轮（原型快速验证）：
- 材料推荐：PETG 或 PA12（尼龙 12）
- 打印参数：层高 0.16mm，填充 100%
- 关键特征：
  - 中心孔：phi8mm（与 AK80-9 输出轴适配，H7 公差）
  - 外侧槽宽：4.5mm（适配 4mm OD Bowden 线缆）
  - 外侧槽深：3mm（线缆嵌入后不凸出）
  - 线缆固定孔：phi2mm 侧孔，用于穿线后打结
  - 紧定螺钉孔：M4，用于锁定在电机轴上

如 CNC 铝合金绕线轮：
- 材质：6061-T6 铝合金
- 中心孔：phi8mm H7
- 侧面 M4 紧定螺钉 2 个（间隔 90deg）
- 表面本色阳极氧化或黑色阳极氧化

#### 3.2 绕线轮安装到电机

1. 清洁电机输出轴（phi8mm）和绕线轮中心孔，使用 IPA。
2. 将绕线轮滑入电机轴，调整位置使线缆槽对齐线缆出口方向。
3. 在 M4 紧定螺钉螺纹上涂 Loctite 243。
4. 拧入紧定螺钉，扭矩 1.5 Nm（使用扭力螺丝刀）。
   - 2 个螺钉间隔 90deg，交替拧紧以确保同心。
5. 用手旋转电机轴，确认绕线轮无轴向窜动，无偏心。
6. 如使用键槽配合：安装键销（4x4mm 钢键）+ 轴向卡簧。

#### 3.3 预绕线（线缆终端准备：见 Step 4）

- 在 Step 4 完成线缆终端处理后，将线缆末端穿过绕线轮侧面的 phi2mm 固定孔。
- 拉紧线缆，使终端（钢珠或铝压接头）卡在孔外侧。
- 手动将线缆在绕线轮上紧密排绕 3-5 圈（不可重叠）。
- **说明：** 最终绕线圈数由调试阶段确定。原型阶段先绕 3 圈作为起点。

#### 3.4 电机安装架

- 电机安装架为 CNC 6061 铝合金 L 形支架。
- 使用 M4 x 12mm 螺钉将电机固定到安装架。
  - 4 颗螺钉，Loctite 243，扭矩 2.0 Nm。
- 将安装架固定到控制盒侧壁（或底部）：
  - 使用 M3 x 8mm 螺钉，扭矩 0.8 Nm。
  - 两颗对角螺钉先预紧，再全部锁紧。

**验证点：** 绕线轮转动顺畅，电机轴无晃动，安装架无倾斜。

### Step 4: Bowden 线缆裁剪与终端处理

**目标：** 准备 2 根 Bowden 线缆（左右各一），长度精确，终端安全可靠。

#### 4.1 材料确认

- Bowden 线缆：PTFE 内衬自行车刹车线，外径 4mm，内径 1.5mm。
- 钢缆内丝：直径 1.5mm，7x7 或 1x19 编织。
- 线缆长度：1.5m（标准），实际长度需根据用户体型裁剪。

#### 4.2 长度确定

**计算公式：**
```
L = L_waist + L_thigh + 100mm (余量)
```

其中：
- L_waist: 从电机出口到腰带锚点入口的路径距离
- L_thigh: 从腰带锚点到大腿固定点的路径距离

**原型默认长度：**
- 小号 (waist <= 80cm): 1.2m
- 中号 (waist 80-110cm): 1.4m
- 大号 (waist >= 110cm): 1.6m

先全部按 1.5m 裁剪，试穿后调整（Step 8 确定最终长度）。

#### 4.3 切割与终端处理

```
    +-------- Bowden 线缆结构 --------+
    |                                  |
    |  +------+                        |
    |  | 钢丝  |  内丝 (1.5mm钢缆)    |
    |  | --->  |                        |
    |  | PTFE  |  PTFE 内衬 (减小摩擦) |
    |  |<----->|                        |
    |  | 钢带  |  螺旋钢带 (抗压)     |
    |  |<----->|                        |
    |  | PVC   |  外皮 (保护)          |
    |  |<----->|                        |
    |  +------+                        |
    +----------------------------------+
```

**终端步骤（每根线缆两端都要处理）：**

1. **切割外皮：** 用剥线钳在距端部 30mm 处切断 PVC 外皮，抽出 30mm 内钢丝。
2. **切割钢带：** 用剪切钳剪掉露出的螺旋钢带，注意不要损伤 PTFE 内衬。
3. **切割 PTFE 内衬：** 用锋利刀片切掉 PTFE 内衬，露出纯钢丝 30mm。
4. **钢丝端部处理：**
   - **方法 A（推荐）：铝压接头。** 将 1.5mm 钢丝穿入铝压接头（phi3mm 外径 x 8mm 长），用压线钳压紧 3 次。
   - **方法 B（快速原型）：钢丝端部焊接**。将 10mm 钢丝端部用焊锡浸润，形成 phi3mm 焊球。注意：焊锡连接强度较低，仅用于原型初期。
   - **方法 C（最安全）：钢珠终端。** 使用自行车刹车线专用钢珠终端，用专业压线工具压接。
5. **热缩管保护：** 将 phi4mm 热缩管套在终端与线缆交接处，热风枪 150C 收缩。
6. **拉力测试：** 用弹簧秤或拉力计拉至 200N 保持 10 秒，确认终端不脱落。

#### 4.4 线缆标识

- 在线缆两端贴标签：`L-LEFT` / `L-RIGHT`（左/右）。
- 在距终端 500mm 和 1000mm 处贴标识点，方便后续长度调整。

**验证点：** 终端的最大承受拉力 >= 800N（机械熔断器限值），钢缆无散股，热缩管包裹紧密。

### Step 5: 碳纤维髋部锚点加工与预装配

**目标：** 将 2mm 碳纤维板 CNC 切成的髋部锚点加工完成，准备缝入压缩衣。

#### 5.1 来料检查

- 材质：预浸碳纤维板 2mm 厚（单向或编织，强度要求 > 200 MPa）
- CNC 切割完成，边缘光滑
- 每片包含以下特征：
  - 2 个 phi6mm 安装孔（用于连接 Bowden 线缆终端）
  - 4 个 phi3mm 缝制孔（用于固定到压缩衣布料）
  - 2 个 phi4mm 定位孔（用于定位模板）
  - 弧形轮廓贴合髋部

```
    碳纤维锚点俯视图 (单侧):

     +-----------------------------+
     |  ---  ---                  |
     | | o  | o  |   <- phi6mm 安装孔|
     |  ---  ---                  |
     |   o       o                |
     |   (定位孔 phi4mm)          |
     |                             |
     |  o    o    o    o          |
     |  缝制孔 phi3mm (4个)       |
     |                             |
     +-----------------------------+
           | 布料方向
   (弧面贴合髋骨外侧)
```

#### 5.2 边缘处理

- 用 200# -> 400# -> 800# 砂纸湿磨边缘，消除所有锐边。
- 检查有无层间分离（delamination）。如有则废弃该片。

#### 5.3 钻孔（如 CNC 已打好则跳过）

- 使用 3mm 铝板钻孔模板精确定位。
- 使用碳纤维专用钻头（硬质合金或 PCD 涂层）。
- 钻孔参数：
  - 转速：10,000-15,000 RPM
  - 进给：慢速、均匀
  - 垫板：在碳纤维下方垫木板防止出口撕裂
- 钻孔后用压缩空气吹净粉尘。

#### 5.4 表面处理

- 用 IPA 彻底清洁表面，去除脱模剂残留。
- 如需要粘接尼龙织带（增强缝制强度），在相应区域用 200# 砂纸轻微打磨，增加粘接力。
- 用 IPA 再次清洁。

#### 5.5 预装配（支架 + 锚点）

**注意：** 此步骤在与压缩衣集成之前完成，确保机械接口正确。

- 将 Bowden 线缆输入端的外壳接头（通常为 M6 螺纹端）穿过锚点 phi6mm 孔。
- 用 M6 螺母固定，扭矩 4 Nm。使用 Loctite 243。
- 将线缆的另一端（未固定端）在布线路径上初步走位，确认长度。
- **此时不固定线缆到绕线轮。**

**验证点：** 锚点无裂纹，phi6mm 孔与线缆终端螺母配合顺滑，M6 螺母锁紧后不旋转。

### Step 6: 压缩衣改装 —— 碳纤维锚点集成

**目标：** 将碳纤维锚点牢固缝入压缩衣的腰部/髋部位置。

#### 6.1 锚点定位

- 压缩衣为医疗级尼龙-氨纶混纺面料（弹性 4 向拉伸）。
- 试穿压缩衣，标记髋部大转子（greater trochanter）位置。
- 碳纤维锚点应当位于髋骨外侧正上方（约在腰带位置下方 20-30mm）。
- 左右锚点对称，间距根据用户腰宽调整。

```
    压缩衣锚点定位 (正面):

           +---------------+
           |   压缩衣      |
           |  (Nylon-Spandex)|
           |    +-----+    |   <- 腰带位置
           |    | 锚点|    |
           |    +-----+    |
           |   --------    |   <- 缝线
           |               |
           |   +-----+     |
           |   | 锚点|     |
           |   +-----+     |
           +---------------+
```

#### 6.2 缝制准备

- **加固层：** 在锚点与压缩衣布料之间垫入一块 100x100mm 尼龙加强布（与压缩衣同色）。
- 加强布使用 3M 双面喷胶临时固定到压缩衣内侧。
- 将碳纤维锚点放置在压缩衣外侧，通过 phi3mm 缝制孔标记定位。

#### 6.3 缝制工艺

- **线材：** 高强力尼龙线（#60 或 #80，抗拉 > 50N）或 Kevlar 缝纫线。
- **针：** 工业缝纫机 #18 或 #20 号针（适配碳纤维板 + 厚布料）。
- **针法：**
  - 先沿锚点 4 个 phi3mm 孔各缝 8 字型加固 10 个来回（手工缝或工业机）。
  - 然后沿锚点外轮廓缝 2 圈（距边缘 3mm 和 6mm）。
  - 每 5mm 一针，锁线（backstitch）起始和结束。
- **双面加固：** 压缩衣内侧也缝入第二块加强布，形成三明治结构：
  ```
  外侧：碳纤维锚点
  中间：尼龙加强布 + 压缩衣布料
  内侧：尼龙加强布
  （3 层布料 + 1 层碳纤维缝在一起）
  ```
- 缝制完成后，人工拉伸锚点区域 3 次（各方向 50N 拉力），确认无线缝脱落。

#### 6.4 柔性盖板（可选）

- 在碳纤维锚点外侧缝一块弹性尼龙盖布（可掀开），用于遮盖硬质锚点，改善外观和舒适度。
- 盖布使用尼龙搭扣（魔术贴）固定一侧，方便检修。

**验证点：** 锚点牢固，无相对布料位移，缝线无抽丝，拉伸测试后恢复原状。

### Step 7: 压缩衣改装 —— sEMG 电极缝制

**目标：** 将 8 片银镀织物 sEMG 电极缝入压缩衣内的大腿内侧位置。

#### 7.1 电极规格

- 材料：银镀尼龙针织织物（Silver-plated nylon knit）
- 尺寸：每片 30x20mm 矩形（可根据需要裁剪）
- 阻抗：< 1 Ohm/平方（表面电阻）
- 连接：不锈钢微型按扣 (Snap connector)

#### 7.2 电极位置

使用纸板定位模板，确保左右对称。8 个电极位置如下：

| 电极编号 | 位置 | 目标肌肉 |
|----------|------|----------|
| sEMG-01 | 左大腿内侧前 1/3 | 股内侧肌 (Vastus Medialis) |
| sEMG-02 | 左大腿内侧中 1/3 | 半腱肌/半膜肌 |
| sEMG-03 | 左大腿内侧后 1/3 | 股二头肌长头 |
| sEMG-04 | 左大腿内侧参考点 | 骨骼附近（无肌肉） |
| sEMG-05 | 右大腿内侧前 1/3 | 股内侧肌 |
| sEMG-06 | 右大腿内侧中 1/3 | 半腱肌/半膜肌 |
| sEMG-07 | 右大腿内侧后 1/3 | 股二头肌长头 |
| sEMG-08 | 右大腿内侧参考点 | 骨骼附近（无肌肉） |

```
    大腿横截面 (俯视):

           前侧 (Quadriceps)
          +---------------+
          |               |
    内侧  |   股骨 (骨)    |  外侧
     <---+  -----------  +--->
          |  VM  ST  BF  |
          |  1   2   3   |
          |  4 (参考点)  |
          +---------------+
           后侧 (Hamstrings)

    1 = sEMG-01/05: 股内侧肌
    2 = sEMG-02/06: 半腱肌/半膜肌
    3 = sEMG-03/07: 股二头肌
    4 = sEMG-04/08: 参考
```

#### 7.3 缝制步骤

1. **标记：** 用户穿上压缩衣，在站立位下，触诊定位肌肉位置，使用水溶性标记笔在布料表面标记电极中心点。
2. **裁剪：** 在标记点处裁剪 5mm 小孔（用于安装按扣公端）。
3. **按扣安装：**
   - 在布料内侧安装不锈钢按扣的公端（male snap）。
   - 在银镀织物电极中心安装按扣母端（female snap）。
   - 将电极通过按扣固定到布料内侧。
4. **缝制固定：**
   - 使用银导电缝纫线（如 Shieldex 或同等）将电极四角缝在布料上。
   - 每角缝 5 针，确保导电接触良好。
   - 缝纫线与按扣之间也需连接（可选：点少量银环氧导电胶增强）。
5. **布线：**
   - 每根电极通过导电缝纫线或极细漆包线（AWG 32）引至腰部连接器。
   - 布线路径沿压缩衣内部缝线走向，避免拉伸时断裂。
   - 使用热熔胶薄层固定导线的关键弯折点。
6. **连接器：**
   - 在压缩衣腰部内侧安装 10 针（或 8 针）医用级连接器。
   - 每个电极引脚对应一根导线。
   - 多余的 2 个引脚作为屏蔽地（GND）。

#### 7.4 绝缘与保护

- 在电极的背面（接触布料的那一面）用非导电织物覆盖，防止电极之间短路。
- 在连接器区域周围涂一层薄的医用硅胶（施加后室温固化 24h），提供 IPX4 防水。

**验证点：** 每对电极间的电阻 < 10 Ohm（用万用表测量）；施加拉伸后电阻变化 < 20%；按扣连接无松动。

### Step 8: Bowden 线缆布线路径与固定

**目标：** 从控制盒（腰后）经过路由，将 Bowden 线缆引导至髋部锚点。

#### 8.1 布线路径总览

```
                  +----------+
                  | 控制盒    | <- 腰后 (L5/S1 位置)
                  | (电机)    |
                  +-----+----+
                        |
               Bowden 线缆 (左右各1)
              +---------+---------+
              |         |         |
              v         v         v
         +--------+ +--------+ +--------+
         | 左侧   | | 右侧   | | 布线   |
         | 腰带引导| | 腰带引导| | 夹扣   |
         +----+---+ +----+---+ +--------+
              |          |
              v          v
         +--------+ +--------+
         | 左髋   | | 右髋   |
         | 碳纤   | | 碳纤   |
         | 锚点   | | 锚点   |
         +--------+ +--------+
```

#### 8.2 布线引导夹

使用 3D 打印 TPU 柔性夹扣（BOM 项目 #9）将 Bowden 线缆固定在腰带路径上。

夹扣安装位置（每侧）：

| 夹扣 | 位置 | 功能 |
|------|------|------|
| #1 | 控制盒出口 50mm | 固定线缆初始方向 |
| #2 | 腰带侧面（髂前上棘上方） | 引导线缆转向大腿 |
| #3 | 腰带正面中间（可选） | 如线缆交叉走位 |

#### 8.3 弯曲半径控制

- Bowden 线缆的最小弯曲半径：**50mm**（工程架构文档要求）。
- 在布线路径中加入 phi50mm 的预成型弯管或 3D 打印导向块来强制保证最小半径。
- 急弯处（如控制盒出口 90deg 转向）必须使用半径 >= 50mm 的弯头。

```
  错误:                   正确:
  |                      |
  |   \                  |    +-
  |    \                 |   /
  |     \  < 弯曲太小    |  /  半径 >= 50mm
  |      \               | /
  v       v              vv
```

#### 8.4 线缆长度最终调整

- 在完成布线路径固定后，用户穿上压缩衣。
- 线缆一端连接绕线轮，另一端连接锚点。
- 在髋关节最大屈曲位（坐姿，膝盖向胸部靠拢）下，检查线缆是否过紧。
- 在髋关节最大伸展位（站姿后伸腿）下，检查线缆是否过松。
- 调整长度：在锚点端增加一个调整螺丝扣（turnbuckle）或改变绕线轮的固定位。
- **标记最终线缆长度**，在非锚点端裁剪多余长度。

#### 8.5 布线验证

- 在所有固定点安装完成后，手拉线缆确认全线畅通无卡滞。
- 用户做全范围动作（站立->坐->行走->弯腰），确认线缆不钩挂、不摩擦异常。

**验证点：** 线缆路径无急弯（半径 >= 50mm），无摩擦尖锐边缘，各固定夹扣牢固。

### Step 9: 总装 —— 电机-线缆-锚点连接

**目标：** 将预处理好的组件最终连接为完整系统。

#### 9.1 控制盒合盖

- 检查控制盒内部所有连接：
  - 电池输出线 -> 主板的电源输入（XT60 连接器，已锁紧）
  - 电机线 -> 驱动板输出（3 相，标记 U/V/W）
  - sEMG 连接器 -> 主板 ADC 输入（已有引脚定义）
  - BLE 天线 -> 陶瓷天线或外置天线（如使用外置）
- 确认所有连接器有防松脱设计（锁扣或螺纹）。
- 放置密封 O 型圈。
- 合上上壳，用 4 颗 M3 x 10mm 螺钉固定。
  - 螺钉螺纹涂 Loctite 243。
  - 对角交替拧紧，扭矩 0.8 Nm。
- 在壳体接缝处贴上封条（原型阶段使用易碎贴，防止非授权开盖）。

#### 9.2 控制盒安装到腰带

- 使用 M5 x 12mm 螺钉将腰夹安装到控制盒底部。
  - 4 颗螺钉，Loctite 243，扭矩 3.0 Nm。
- 将腰夹滑入腰带上的兼容轨道或织带环中。
- **注意：** 控制盒中心应对齐用户腰椎 L5/S1 位置（后腰正中）。

#### 9.3 线缆连接

- 将 Bowden 线缆的绕线轮端连接到绕线轮。
  - 穿入侧孔，打结或用压接头固定。
  - 确保钢缆在绕线轮的起始槽内。
- 将线缆另一端的 M6 螺纹端穿过锚点的 phi6mm 孔，用 M6 螺母固定。
  - 扭矩 4 Nm。
  - 使用 Loctite 243 防松。

#### 9.4 机械熔断器验证

- 确认线缆上串联了 800N 机械熔断器（在终端附近）。
- 手动拉线确认熔断器安装方向正确（拉紧时受力）。

#### 9.5 保护罩安装

- 在绕线轮区域安装透明亚克力防护罩。
- 使用 M3 x 6mm 螺钉固定，扭矩 0.4 Nm。

**验证点：** 所有机械连接紧固，线缆终端牢固，防护罩安装到位。

### Step 10: 最终质量检查

#### 10.1 机械完整性检查

- [ ] 所有螺钉就位，无缺失
- [ ] 所有弹簧垫圈已安装
- [ ] 螺纹锁固胶（Loctite 243）已用于所有金属-金属连接
- [ ] 扭力规格已满足（见第 5 节）
- [ ] 无锐边外露
- [ ] 控制盒防水接缝密封完整

#### 10.2 线缆张力测试

使用张紧测试台（自制）：

1. 在锚点端连接弹簧秤或数字拉力计（量程 >= 1000N）。
2. 手动旋转绕线轮，直至线缆刚好拉直（预紧力 ~20N）。
3. 用电机缓慢施加张力，记录以下数据：
   - 线缆张力从 0 -> 200N -> 0（回程）的力-位移曲线
   - 回程迟滞（应 < 20N）
   - 在 200N 下保持 30 秒，记录张力衰减（应 < 5%）
4. 记录结果，与工程接受标准对比：
   - 最大摩擦：< 50N（在 200N 张力下）
   - 无卡滞或急跳

#### 10.3 电极阻抗检查

- [ ] 使用万用表或 LCR 表测量：
  - 每对 EEG 电极间：< 10 Ohm 直流电阻（头带组装后）
  - 每对 sEMG 电极间：< 5 Ohm 直流电阻
  - 所有电极对地绝缘：> 1 MOhm（50V 偏压）
- [ ] 电极按扣连接正常，无松动

#### 10.4 贴合度测试（用户试穿）

用户穿上完整的系统，检查：

- [ ] 压缩衣贴合度：紧身但无压迫点，不影响呼吸
- [ ] 髋部锚点位置正确（对准大转子）
- [ ] Bowden 线缆在全范围运动（坐下、弯腰、行走）中不钩挂
- [ ] 控制盒位置舒适，坐下时不会顶到椅背
- [ ] EEG 头带在运动时不滑脱
- [ ] sEMG 电极在大腿内侧不滑动
- [ ] 总系统重量 < 2.5 kg

#### 10.5 功能检查（不通电）

- [ ] 电机轴手动旋转：无卡涩、无异响
- [ ] 绕线轮旋转：无偏心、无轴向窜动
- [ ] Bowden 线缆在护套内自由滑动：无卡滞
- [ ] 控制盒螺钉全部拧紧，无松动
- [ ] 电池组在壳体内无晃动

#### 10.6 记录

将所有检查结果记录在 **质量控制检查清单**（第 6 节）中，签字确认。

- 所有扭矩值在干燥螺纹（无润滑）条件下给出。

## 8. 附录 B: 首次通电前检查

在完成全部机械组装后，首次通电测试前，按以下顺序检查：

### 8.1 机械检查（已完成 Step 10）

- [ ] 所有 QC 项目通过
- [ ] 所有紧固件扭矩达到规格
- [ ] 保护罩已安装

### 8.2 电气检查

- [ ] 电池电压：21.0-25.2 V（用万用表测 XT60 连接器）
- [ ] BMS 输出端电压：与电池电压一致
- [ ] 主板上电源指示灯（如有）在通电后亮起
- [ ] 电机三相线电阻：每两相之间应相等（~0.1-0.5 Ohm）
- [ ] 电机三相线对地绝缘：> 1 MOhm

### 8.3 安全检查

- [ ] 紧急停止按钮（硬件 E-Stop）安装到位
- [ ] E-Stop 按下后电机电源断开确认
- [ ] 线缆机械熔断器存在且方向正确
- [ ] 电池防火袋随手可及
- [ ] 灭火器（CO2 或干粉）在工作区 3m 范围内

### 8.4 通电顺序

```
1. 安装电池（最后一步连接 XT60）
2. 听声检查：无异常爆音、无烟、无异味
3. 测量电压：确认各电源轨正确（3.3V, 5V, 12V/24V）
4. 关闭系统
5. 装回控制盒盖
6. 第二次通电：全系统启动
7. 确认 BLE 连接：头带 -> 手机、控制盒 -> 手机
8. 手动旋转电机（低功率模式）：确认方向正确
9. 加载最小张力（~20N）：确认线缆平稳
10. 渐变至目标张力（200N）：监控系统温度
```

**注意：** 首次通电必须在 ESD 安全工作台上进行，旁边有人监护。如发现异常立即按下 E-Stop。
