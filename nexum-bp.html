1. 我们在做什么

逐念而行构建从人类神经意图到物理世界动作的实时映射层。首款硬件载体 Nexum One 神经意图外骨骼，面向脊髓损伤和卒中偏瘫患者——大脑运动意图完整、但神经信号无法到达肌肉的人。

现有外骨骼的控制方式是被动的：按钮（Ekso）、倾斜传感器（ReWalk）、肌电（HAL）。这些信号产生在运动意图之后。Nexum选择非侵入EEG路线——在动作发生前200-500ms，从头皮脑电中检测运动准备电位（MRCP），提前预判使用者即将执行的动作。

核心技术架构

Nexum One分为离线训练和在线运行两条时间线：

离线训练（GPU服务器）：柔性肌腱的神经力学全身肌骨模型（416 MTU, Hill-type）仿真人体步行。RL策略在此环境中训练，学习从步态相位到关节力矩的映射函数。训练完成后经策略蒸馏压缩为轻量网络（<5MB），部署至嵌入式设备。肌骨模型本身不离开服务器，不进入实时控制回路。

在线运行（嵌入式设备）：EEG检测运动意图（四类：行走/停止/起立/无意图），IMU检测步态相位。RL策略接收这两组信号，输出关节力矩指令→直驱电机执行。EEG失效时系统无缝降级至纯IMU模式。

肌骨模型与EEG不直接对接。RL策略是两者之间的桥梁——在高精度MTU级仿真中训练，部署后以EEG+IMU作为输入进行推理。

2. 为什么过去没人做成、为什么是现在

非侵入EEG+下肢外骨骼从未被商业化。四个障碍在2025-2026年首次被突破：

障碍一：EEG步行信噪比的物理极限

运动伪迹与神经信号共享相同频带。Porr et al.（PLOS ONE 2023）提出"SNR-wall"理论——噪声超过阈值后任何算法都无法恢复信号。这是物理规律，不是工程问题。2024-2025年MNAE微针干电极突破：接触阻抗降至2.6kΩ@10Hz，信号幅度超湿电极39-44%（Materials Today Nano 2026），有源干电极SNR达1.27dB，与湿电极1.30dB几乎无差。干电极的信号质量瓶颈在物理层面首次被突破。

障碍二：跨学科团队极度稀缺

需要同时掌握EEG信号处理（神经科学）、外骨骼控制（机器人学）、嵌入式系统（EE）、临床康复（医学）——四种人才不在同一圈子。中国2025年才成立全国脑机接口教育联合体（天津大学牵头14所高校），培养周期5-8年。

障碍三：监管真空

BCI和外骨骼各有独立监管分类，组合产品的审评路径在NMPA/FDA/CE均无先例。中国2025年6月才发布首个外骨骼行业标准YY/T 1973-2025（2026年7月实施），2025年3月才首次为BCI单独立项（国家医保局）。在此之前没有审批通道可走。

障碍四：资金真空

三家上市外骨骼公司（Ekso, ReWalk, Cyberdyne）市值长期低于$300M且从未盈利。纯BCI公司还在早期阶段。中国BCI+外骨骼赛道单笔融资不超过1亿元量级，远不够覆盖硬件+软件+临床认证的全周期成本。没有可参照的成功商业化案例。

为什么Nexum现在切入

上述四个障碍在2025-2026年首次同时松动：干电极物理突破+QDD电机扭矩密度突破+中国11省外骨骼医保+BCI定价落地+NMPA绿色通道加速。此外，Nexum自建了柔性肌腱的神经力学全身肌骨仿真环境（416 MTU, Hill-type），作为RL策略的高精度训练环境——仿真精度直接决定部署策略的性能上限。

我们的四个战略选择

非侵入EEG，而非侵入式BCI或肌电。侵入式（Neuralink/Synchron）需要开颅手术，患者基数极小。肌电（HAL）受限于深层肌肉无法测量。非侵入EEG无手术风险，且提供200-500ms的运动预判窗口。混合感知架构（EEG预测+IMU主控）规避了对单一信号源的可靠性依赖。

下肢，而非上肢。上肢EEG康复已有g.tec、赛博灵科等商用产品。下肢市场更大（1500万 vs 800万患者），暂无直接竞争者，支付路径更明确（11省外骨骼医保+CMS K1007均覆盖下肢）。

医疗，而非消费。消费外骨骼（极壳Hypershell等）面向健康人群增强，无医保覆盖，需自费购买。医疗外骨骼有医保支付方，且患者支付意愿远高于健康人群的"可选消费"。

服装式，而非框架式。框架式外骨骼（Ekso/ReWalk/程天）重15-25kg，为完全瘫痪患者设计，仅限医院使用。服装式（~2.5kg）面向能站立但走不好的不完全瘫痪患者，可居家独立使用。患者基数从100万（完全性SCI）扩展到1500万（不完全性SCI+卒中偏瘫）。

4.1 工作原理

  1. 大脑生成MRCP（运动准备电位）——自主运动前200-500ms出现
  2. EEG-Sense 意图感知头带检测（8通道，250SPS，EEGNet分类）
  3. BLE传输至腰部 NeuroCore 控制中枢
  4. 神经意图映射层：EEG意图 + IMU步态相位 → 最优力矩曲线
  5. 髋两侧直驱电机执行辅助力矩（12.5Nm峰值，5Nm持续）
  6. 力传感器反馈 → PID控制回路确保安全

5.2 神经感知层：EEG + IMU混合架构

EEG（预测性意图检测）

  • 8通道干电极，250SPS，24-bit ADC（TI ADS1299）
  • 检测目标：运动准备电位（MRCP），自主运动前200-500ms出现
  • 算法：EEGNet（2018，Lawhern et al.）— 仅约2000参数的紧凑CNN，适合嵌入式部署
  • 分类任务：{行走, 停止, 起立, 无意图} 四类，单次检测准确率70-85%（实验室条件）
  • 信号处理链：共平均参考(CAR) → 0.5-45Hz带通 → 50Hz陷波 → EEGNet推理 → 指数滑动平均平滑

IMU（主力运动感知）

  • 4个6轴IMU（BMI270），分布于大腿与小腿，100Hz更新率
  • 功能：步态相位检测（着地/支撑/摆动）、关节角度估计、跌倒检测
  • IMU为主控制信号源——当EEG分类置信度低或信号质量差时，系统无缝降至纯IMU模式

为什么混合：EEG提供"提前预判使用者即将迈步"的预测能力（比纯IMU快200-500ms），IMU提供"精确判定当前步态相位"的实时精度。两者互补，而非替代。当EEG失效时，IMU独立工作——安全无影响。

5.3 意图解码层：离线训练 + 在线推理

意图解码层分为离线训练和在线推理两条时间线，肌骨模型和EEG不直接对接——RL策略是两者之间的桥梁。

离线训练（GPU服务器）

  • 已构建柔性肌腱的神经力学全身肌骨模型（416 MTU, Hill-type），可仿真人体步行
  • RL策略在该高精度仿真环境中训练，学会"不同步态相位需要多少关节力矩"
  • 训练完成后策略网络蒸馏为轻量模型，部署至嵌入式设备。肌骨模型本身留在服务器，不进入实时控制回路
  • 当前状态：已验证裸肌骨建模与步行仿真。需扩展至包含外骨骼动力学+混合穿戴结构的联合仿真环境

在线推理（嵌入式设备 — NeuroCore）

  • 意图分类器：EEGNet（~2000参数），输入8通道×250Hz EEG（100ms滑动窗口），输出四类运动意图
  • RL策略网络：接收EEG意图分类结果 + IMU步态相位，输出关节力矩指令
  • 部署：ONNX Runtime，INT8量化，嵌入式推理 <5ms（EEGNet）+ RL策略前向
  • EEG失效时策略降级为纯IMU模式运行
  • V1阶段策略离线预训练，不做在线学习。V2阶段引入联邦学习更新

5.4 力控执行层：直驱电机+FOC控制

硬件选型

  • 电机：准直驱电机（参考 CubeMars AKE60-8 或同级别），设计目标单侧≤0.5kg，峰值扭矩≥12.5Nm，持续扭矩≥5Nm
  • FOC控制+驱动：TMC4671（硬件FOC控制器）+ DRV8316（三相栅极驱动），SPI接口
  • 主控芯片（位于NeuroCore控制中枢）：STM32H743（Cortex-M7, 480MHz），运行FreeRTOS，负责实时FOC控制与传感器融合。AI推理：Hailo-8L NPU（13 TOPS @ 1.5W），ONNX Runtime框架，执行EEGNet推理与RL策略前向
  • 传感器反馈：14-bit磁编码器（位置/速度）+ 应变片（扭矩估计）

控制架构（分层安全设计）

  1. 内环（1kHz）：FOC电流环 PI控制 — 硬件级实时，不可绕过
  2. 中环（500Hz）：扭矩环 PI控制 — 接收RL策略的力矩目标，经安全限幅后输出
  3. 外环（100Hz）：步态相位检测 + 意图分类 → 力矩查表/RL推理
  4. 安全层：独立看门狗，扭矩硬限幅（<12.5Nm），急停按钮，三级降级（EEG失效→IMU纯模式→安全停止）

5.5 穿戴结构层：碳纤维骨架+弹性织物

NeuroSuit的物理形态——服装式外骨骼的关键差异化载体：

  • 碳纤维骨架：T700级预浸料编织布，Toray医疗级（$50-200/kg, FDA认证）用于患者接触部位，恒神工业级（$9-15/kg）用于非接触结构件。用量约0.5-1.5kg/套。锚定点缝入弹性织物内层
  • 弹性织物基底：运动压缩裤基底（参照2XU/CW-X梯度压缩设计），多点压力分散。碳纤维锚定骨架缝入织物，无刚性外框。可内穿于常规衣物
  • 重量分布：整机~2.5kg——头带80g + NeuroCore 350g + 肌动服~2.1kg（含电机×2 + 碳纤维骨架 + 电池）
  • 与框架式的本质区别：Ekso/ReWalk/程天等框架式外骨骼重15-25kg，刚性外骨骼穿在衣物外。服装式将执行器集成到衣物内部，穿脱<10秒，重心贴近人体，无刚性外框压迫

5.6 策略进化层：离线联邦学习（V2）

V1阶段不包含实时在线学习。V2阶段引入离线联邦学习框架：

  • 每台设备在本地积累使用数据（EEG+IMU+力矩+步态）
  • 定期（每日/每周）上传加密模型梯度至中心服务器
  • 服务器聚合梯度、更新全局策略网络
  • 更新后的策略推送到各设备
  • 原始EEG/步态数据永不离开设备——仅模型梯度上传

5.7 安全架构

作为穿戴式医疗设备，安全设计是技术栈的基础而非附加：

  • 硬件安全：急停按钮独立于主控回路，直连电机驱动使能引脚
  • 软件安全：三级降级：全功能 → IMU纯模式（EEG失效）→ 零扭矩（系统故障）
  • 机械安全：电机扭矩硬限幅 <12.5Nm，关节机械限位 0-45°
  • 数据安全：EEG原始数据本地存储，联邦学习仅上传模型梯度
  • 合规安全：设计遵循IEC 62304（医疗软件）和ISO 13485（质量管理）
