逐念而行

从人类神经意图到物理世界动作的通用映射。

首款产品：Nexum One — 神经意图外骨骼 | 长期愿景：人机共生

1. 我们在做什么

逐念而行聚焦的是人机之间缺失的接口层：非侵入地读取人类神经意图，翻译成物理世界动作。这是人机关系中最后一块空白。

人机共生不是营销词汇。不是因为Licklider在1960年预测了它。因为全行业在解决同一个核心问题：如何缩短从人类产生运动意图到物理动作执行之间的延迟——即意图-动作映射的时效性。Neuralink用侵入式，Meta用肌电腕带，Apple用眼动追踪。逐念而行选择的是技术要求最高但需求最刚性的入口——非侵入EEG + 外骨骼：帮助那些生物通路已经断裂的人。

产品只有一个：Nexum One — 神经意图外骨骼。

2. 为什么过去没人做成

非侵入脑电与外骨骼的组合商业化在此前未出现成功案例。根本原因在于四个必要条件从未同时满足，而非缺乏尝试。这四个条件在2025-2026年首次全部具备。

门槛一：EEG运动信噪比

问题：步行中的EEG信噪比极差。运动伪迹淹没脑电信号。MRCP（运动准备电位，5-30μV）被背景脑电（~100μV）和步态震荡掩盖。

什么变了：EEGNet（2018，仅2000参数）被验证可从带噪EEG中分类运动意图。深度学习去噪方法（2023-2025）将干电极SNR推至44dB。没有单个模型突破，但工程实践的积累使首次达到可分发的可靠性基线。

谁试过：g.tec做上肢EEG康复（recoveriX），不做下肢。Duke/Nicolelis Walk Again Project（2016）用EEG驱动下肢外骨骼证明了原理可行但未商业化。

门槛二：电机扭矩密度与电池能量密度

问题：髋关节助力所需的扭矩（15Nm峰值）与穿戴式设备的重量约束（<3kg）在此前无法同时满足——能输出15Nm的电机重量超过5kg，轻于1kg的电机输出不足5Nm。

什么变了：Hao Su Lab于2025年发布准直驱膝关节外骨骼，单侧电机0.38kg、持续扭矩18Nm、峰值扭矩密度47.4Nm/kg。同年Hypershell X Ultra以2.2kg总重（含电池）实现消费级量产。国产准直驱电机扭矩密度在2024-2025年从~25Nm/kg提升至40-50Nm/kg，达到穿戴式髋关节辅助的工程临界点。

谁试过：Harvard Soft Exosuit（太重，需10kg背包）。Ekso/HAL/ReWalk 15-25kg，仅限临床使用。

门槛三：监管与支付路径

问题：没有任何国家为BCI+外骨骼组合建立监管通道。

什么变了：中国2025年3月首次发布非侵入BCI医疗服务定价¥960-966/次。11省市2024-2025年将外骨骼康复纳入医保。NMPA二类创新器械绿色通道开放。

谁卡住了：Cyberdyne HAL花了10+年才获得日本医保覆盖。Ekso拿到FDA但个人版定价$91,032/台。

门槛四：学术-商业断层

问题：验证了EEG+外骨骼原理的团队（Duke、EPFL、中山大学）是学术实验室——发完论文就转向下一个课题。有能力做产品的公司（Ekso、ReWalk）没有EEG能力。有EEG能力的公司（g.tec、Emotiv）不做外骨骼。

为什么是现在：该交叉点目前处于空白状态，原因在于需同时具备EEG信号处理与外骨骼工程能力的团队，而非技术原理不可行。四个必要条件在2025-2026年首次同时满足，形成了明确的市场进入窗口。

4. 产品：Nexum One

Nexum One由三个物理模块组成。穿戴形态为紧身衣式：碳纤维锚定骨架嵌入高弹性织物，多点压力分散，无需刚性外框。无外置显示屏。一个电源按钮。

EEG-Sense头带 — 运动头带形态，80g，8通道干电极，BLE无线
控制盒 — 120×80×30mm，铝合金，STM32H7，48Wh可更换电池，一个开关
NeuroSuit外骨骼裤 — 碳纤维骨架缝在弹性织物里，髋两侧直驱电机

穿戴体验接近高弹性压缩运动裤。电机不宽于胯。可内穿于常规衣物内。

工作原理

  1. 大脑生成MRCP（运动准备电位）——自主运动前200-500ms出现
  2. EEG-Sense头带检测（8通道，250SPS，EEGNet分类）
  3. BLE传输至腰部控制盒
  4. AI翻译引擎：EEG意图 + IMU步态相位 → 最优力矩曲线
  5. 髋两侧直驱电机执行辅助力矩（15Nm峰值，8Nm持续）
  6. 力传感器反馈 → PID控制回路确保安全

5. 技术架构

5.1 四层技术栈

Nexum One的技术架构分为四层。每一层的技术选型基于当前技术成熟度与安全要求做出，成熟度标注：「产品级」指已有同类技术在商用产品中部署（如IMU步态检测、FOC电机控制）；「验证级」指在同行评审研究中验证可行、但尚未作为系统集成到商用产品中（如EEGNet在嵌入式设备上的实时分类）；「研究级」指基础研究已发表、但工程化与临床验证尚未完成（如联邦学习在外骨骼上的部署）。

5.2 感知层：EEG + IMU混合架构

EEG（预测性意图检测）

  • 8通道干电极，250SPS，24-bit ADC（TI ADS1299）
  • 检测目标：运动准备电位（MRCP），自主运动前200-500ms出现
  • 算法：EEGNet（2018，Lawhern et al.）— 仅约2000参数的紧凑CNN，适合嵌入式部署
  • 分类任务：{行走, 停止, 起立, 无意图} 四类，单次检测准确率70-85%（实验室条件）
  • 信号处理链：共平均参考(CAR) → 0.5-45Hz带通 → 50Hz陷波 → EEGNet推理 → 指数滑动平均平滑

IMU（主力运动感知）

  • 4个9轴IMU（BMI270），分布于大腿与小腿，100Hz更新率
  • 功能：步态相位检测（着地/支撑/摆动）、关节角度估计、跌倒检测
  • IMU为主控制信号源——当EEG分类置信度低或信号质量差时，系统无缝降至纯IMU模式

为什么混合：EEG提供"提前预判使用者即将迈步"的预测能力（比纯IMU快200-500ms），IMU提供"精确判定当前步态相位"的实时精度。两者互补，而非替代。当EEG失效时，IMU独立工作——安全无影响。

5.3 翻译层：从意图到力矩

意图分类器（EEGNet）

  • 输入：8通道×250Hz EEG信号（100ms滑动窗口，步进20ms）
  • 架构：深度可分离卷积 + 时空卷积核
  • 输出：四类运动意图的概率分布
  • 部署：ONNX Runtime，INT8量化，嵌入式推理 <5ms

肌骨模型（离线仿真训练环境）——在GPU服务器上运行，为RL策略提供高保真虚拟训练环境。策略训练完成后，部署到嵌入式设备的是轻量神经网络（<5MB），不包含肌骨模型本身

  • V1阶段：简化髋关节肌骨模型（2-4自由度），基于OpenSim/MyoSuite框架
  • 作用：在仿真环境中为RL策略提供高保真训练环境
  • 运行位置：离线GPU服务器，非嵌入式设备
  • V2阶段：416肌肉Hill型柔顺肌腱模型（MuscleMimic/EPFL框架），用于更高精度的策略预训练
  • 关键澄清：肌骨模型是训练工具，不是实时控制器。策略训练完成后，部署到嵌入式设备的是轻量神经网络（<5MB），不包含肌骨模型本身。

RL策略网络

  • 算法：PPO（Proximal Policy Optimization）或SAC（Soft Actor-Critic）
  • 训练环境：NVIDIA Isaac Gym/Isaac Lab，并行仿真
  • 训练流程：在仿真中训练数亿步 → 策略蒸馏/量化 → 部署至STM32H7
  • 输出的力矩曲线经PID安全限幅后执行——RL输出为建议性，PID为最终控制权威
  • V1阶段：策略离线预训练，在设备上不做在线学习。V2阶段引入离线模型更新机制。

5.4 执行层：直驱电机+FOC控制

硬件选型

  • 电机：准直驱电机（参考T-Motor AK80-9或同级别），设计目标单侧≤0.5kg，峰值扭矩≥15Nm，持续扭矩≥8Nm
  • 驱动器：DRV8316（三相栅极驱动），SPI接口，支持FOC矢量控制
  • 主控芯片：STM32H743（Cortex-M7, 480MHz），运行FreeRTOS
  • 传感器反馈：14-bit磁编码器（位置/速度）+ 应变片（扭矩估计）

控制架构（分层安全设计）

  1. 内环（1kHz）：FOC电流环 PI控制 — 硬件级实时，不可绕过
  2. 中环（500Hz）：扭矩环 PI控制 — 接收RL策略的力矩目标，经安全限幅后输出
  3. 外环（100Hz）：步态相位检测 + 意图分类 → 力矩查表/RL推理
  4. 安全层：独立看门狗，扭矩硬限幅（<15Nm），急停按钮，三级降级（EEG失效→IMU纯模式→安全停止）

5.5 进化层：离线联邦学习（V2）

V1阶段不包含实时在线学习。V2阶段引入离线联邦学习框架（基于pFedAC，Wang/Yang/Su, arXiv:2605.14423）：

  • 每台设备在本地积累使用数据（EEG+IMU+力矩+步态）
  • 定期（每日/每周）上传加密模型梯度至中心服务器
  • 服务器聚合梯度、更新全局策略网络
  • 更新后的策略推送到各设备
  • 原始EEG/步态数据永不离开设备——仅模型梯度上传

5.6 安全架构

作为穿戴式医疗设备，安全设计是技术栈的基础而非附加：

  • 硬件安全：急停按钮独立于主控回路，直连电机驱动使能引脚
  • 软件安全：三级降级：全功能 → IMU纯模式（EEG失效）→ 零扭矩（系统故障）
  • 机械安全：电机扭矩硬限幅 <15Nm，关节机械限位 0-45°
  • 数据安全：EEG原始数据本地存储，联邦学习仅上传模型梯度
  • 合规安全：设计遵循IEC 62304（医疗软件）和ISO 13485（质量管理）

12. 获证之后：壁垒如何持续

NMPA认证是第一道进入壁垒，但并非唯一的护城河。

  1. 临床数据飞轮：每一台在医院使用的Nexum One都在产生真实临床数据。患者步态+EEG+肌力恢复的纵向队列数据，是任何后来者无法在短期内复制的。程天科技在外骨骼领域积累8年，但其数据不含EEG维度。
  2. AI翻译引擎的累积优势：同一个患者使用100次，AI的策略已经完全不同。出院带回家的患者继续使用，数据持续反馈到仿真训练中。这是软件护城河——硬件可复制，数据不可。
  3. 品牌认知：全球第一款EEG下肢外骨骼。这个品类一旦建立，后来者将面对品类定义者的先发优势。
  4. 全球扩展路径：取得中国NMPA认证后，利用已有临床数据申请FDA 510(k)和CE MDR。结合中国临床数据体量与制造成本结构，具备全球定价优势。
