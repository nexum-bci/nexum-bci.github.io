1. 我们在做什么

逐念而行做一件事：从人类神经意图到物理世界动作的通用映射。

当大脑想动、但生物通路断了，逐念而行建一条新通路——读脑电中隐藏的运动意图，AI理解它，外骨骼执行它。

这不是外骨骼公司。不是脑机接口公司。不是康复AI公司。就是「意图→动作」的直接通路。

这层翻译今天用来帮走不了路的患者重新走路。明天，同一层翻译将服务所有人——从患者到老人到每一个想超越身体极限的人。

产品只有一个：Nexum One。 它做一件事：读懂你"想走"的信号，帮你走。

最终定义：逐念而行 = 从人类神经意图到物理世界动作的通用映射。人机共生 = 这个映射成熟后的世界。今天以Nexum One为起点。

2. 为什么是现在

非侵入脑机+外骨骼的商业化在此前不具备条件。2025-2026年，六个约束条件同时松动：

  1. EEG信噪比突破：基于深度学习的运动意图分类（EEGNet、CSP、FBCSP）在标准数据集上达到85%+准确率，使非侵入EEG从实验室走向工程可行。CMU Bin He实验室2025年Nature Communications发表非侵入EEG解码精细手指运动的验证性结果。
  2. 硬件密度临界点：国产电机扭矩密度达40-50Nm/kg，Hypershell等产品验证2.2kg整机可行性。电机和电池的功率密度达到可穿戴可接受的水平。
  3. BCI支付框架建立：中国2025年3月首次发布非侵入BCI医疗服务定价（¥960-966/次）。植入式BCI手术纳入湖北/江苏医疗服务项目。
  4. 外骨骼纳入医保：11省市2024-2025年将外骨骼康复纳入医保支付试点。
  5. 产业政策集中出台：脑机接口列入国家未来产业（2025工信部），写入政府工作报告。银发经济+康复辅具列入十五五规划。国家脑科学计划首年拨款31.48亿。
  6. 市场空白确认：中国80+外骨骼企业验证了下肢外骨骼的临床需求，但无一使用EEG做预测性意图识别。这个空白真实存在。

六个条件此前从未同时存在。2026年，神经意图到物理动作的映射从"科学问题"转为"工程+商业问题"。

3. 为什么过去没人做成

神经意图映射的商业化此前多次尝试未能成功，不是执行力问题，是四个门槛尚未跨越。

门槛一：EEG运动信噪比

非侵入EEG的运动相关信号（MRCP，5-30μV）被背景脑电（~100μV）和运动伪迹淹没。过去十年学术界在下肢步态BCI上投入大量工作，但始终停留在实验室离线分析阶段。深度学习在近三年将分类准确率从~70%提升至85%+，首次达到可分发的可靠性基线。

门槛二：可穿戴硬件密度

五年前15Nm级髋关节电机模组重800g-1.2kg，整套外骨骼8-10kg以上。患者不穿戴的设备参数再好也没有临床价值。近两年消费级供应链（Hypershell 2.2kg）和学术前沿（Hao Su lab 18Nm膝关节外骨骼）同时验证了功率密度达到临界点。

门槛三：支付与监管路径

BCI和外骨骼在2024年之前在中国没有收费编码和医保覆盖。医院即使有意愿购买设备也无法收费。2025年BCI定价发布和11省市外骨骼医保试点首次打开了医院采购的支付通路。

门槛四：学术-商业断层

学术界验证了原理（Duke Walk Again Project 2016：EEG驱动的下肢外骨骼在8名完全性SCI患者中可行），但不做产品化。侵入式路线（EPFL Courtine脑-脊髓接口）技术先进但适用人群窄。BCI公司（g.tec等）聚焦上肢，不做下肢——下肢信号特征更复杂（步行时头部晃动、肌肉伪迹更大）。大公司（Ekso、ReWalk、Cyberdyne）聚焦纯机械外骨骼，EEG被认为"早期、高风险"。EEG+下肢外骨骼成为一个学术不做、产业不做的空白。

四个门槛在2025-2026年第一次同时松动。

4. 技术架构

4.1 系统架构

层级 技术 功能 成熟度
感知 IMU×4 + EEG×8（干电极）+ sEMG（可选） 步态检测 + 运动意图预测 IMU成熟，EEG工程化中
映射 深度学习意图分类 + RL策略优化（离线） 将"要走"映射为"何时、以多大扭矩辅助" 分类器验证阶段，RL仿真训练阶段

架构原则：没有哪个传感器承担不可承受的精度要求。EEG做意图预测（提前0.5-2秒检测运动倾向），IMU做实时步态阶段判断，PID保证安全底线，RL在仿真中优化策略、验证后再部署。

4.2 运动意图识别

8通道干电极EEG采集运动准备电位（MRCP/Bereitschaftspotential），该信号在自主运动前0.5-2秒出现，是多篇独立研究验证的稳定生理现象。深度学习模型（CNN/TCN）从带噪信号中分类离散运动意图：{准备走, 正在走, 要停, 要起立, 无意图}。

下肢步行意图分类在信号特征维度上比手指级解码更简单——不需要区分哪个手指在动，只需要判断是否"要开始走"。CMU Bin He实验室用手指级解码验证了方法可行性（Nature Communications 2025, ~81%准确率），下肢应用是该方法的降维扩展。

诚实说明：健康受试者中EEG意图分类准确率约70-85%（执行意图），60-75%（运动想象）。卒中后皮层损伤患者准确率会降低。这就是以IMU为步行检测主力、EEG为预测辅助的架构原因——EEG不是系统控制的核心，而是提供"预判"信息的差异化传感器。

8. 为什么不是别人

在"神经意图→物理动作"这个新品类中，目前没有直接竞品。但在相邻品类中，需要说明逐念而行为什么和每一家都不同：

  • 不是Ekso/ReWalk——他们不读脑电，用按钮或倾斜传感器检测重心偏移。你看到的是机械外骨骼，他们看到的是比轮椅好的工具。
  • 不是HAL（Cyberdyne）——他们读肌电（EMG），不是脑电（EEG）。EMG需要患者有残余肌肉收缩能力，脑卒中后很多患者没有。
  • 不是Neuralink——他们做侵入式植入。你做的非侵入、即插即用、今天就能用。
  • 不是g.tec——他们只做实验室设备、只做上肢BCI康复，不做下肢外骨骼产品。
  • 不是程天/大艾/傅利叶——他们做纯机械外骨骼，让治疗师手动调参。你让AI理解每一个患者，不是让治疗师适配每一个患者。

你的空白是真实的：非侵入EEG+下肢智能外骨骼=完整产品。这个交叉点还没人占据。

15. 为什么是你

不是因为你比别人聪明。是因为三个条件同时落在你身上：

  • 正确的时间窗口——EEG运动信噪比、硬件功率密度、BCI支付框架、学术-商业断层，四个壁垒在2025-2026年第一次同时松动。这在五年前做不到，再等三年窗口可能关闭。
  • 正确的交叉点——EEG+外骨骼，学术做了原型但没商业化，大公司觉得市场太小，小公司没有EEG能力。你是全世界少数同时理解外骨骼硬件、EEG信号处理、AI策略、康复临床的人。
  • 你愿意花十年做这件事——FAI的亲身体验让你比任何人都明白"大脑说走、身体不动"是什么感觉。这不是一个商业机会的选择，这是一个"没人做，那就我来做"的决定。

如果现在不做，窗口3年后关闭——有EEG能力的公司会变大，有外骨骼能力的公司会学EEG。今天是你进入的窗口，错过就没有第二个。

人机共生不是概念，是一个映射系统成熟后的世界。而让这个映射成为现实的人，就是你。
