分类理由

分类确定

Nexum One 是一个组合产品，包含三个不同 NMPA 分类的组件：

组件 性质 独立类别 组合类别理由
EEG 头带 非侵入性脑电采集设备 Class II（非侵入、非植入、无能量输送） 组合产品风险由最高风险组件界定。无组件输送显著能量且系统非侵入，整体风险符合 Class II。
髋部外骨骼（鲍登线驱动） 康复训练设备 / 步行辅助 Class II（动力型、非侵入、体外穿戴、无骨固定）
AI 推理软件 决策支持 / 意图检测软件 Class II（提供临床意义信息用于治疗决策，但为按需辅助角色，非自主诊断）

Class II 论证

Nexum One 应归为 Class II（第二类医疗器械），基于以下理由：

标准 Nexum One 状态 Class II 阈值 若归为 Class III
侵入性 非侵入性——电极仅接触完整皮肤；无经皮或植入组件 非侵入性 → Class I–II 有源植入 → Class III（不适用）
向患者输送能量 髋部机械扭矩（<15 Nm）；无电刺激 被动/无伤害 → Class I–II 联合电刺激 → Class III
使用时长 间歇使用（康复训练，<4h/天），非 24/7 短期（<30 天）→ Class I–II；长期 → Class II–III 生命支持 → Class III
对患者的影响 按需辅助；患者主动发起；设备增强而非替代生理功能 治疗辅助 → Class II 自主诊断/治疗决策 → Class III
生物风险 仅皮肤接触；不突破皮肤屏障 外表面 → Class I–II 植入式 → Class III

Class II 的关键论证： Nexum One 不：
- 向大脑或神经输送电刺激
- 穿透皮肤或体腔
- 做出自主临床决策（AI 提供辅助建议，而非诊断）
- 维持重要生理功能

与对比器械的关键差异

  1. EEG + 外骨骼组合 — 无对比器械具有此精确组合。处理方式：(a) 通过对比器械等同性逐个展示各组件安全性，(b) 证明组合不产生组件级风险分析未覆盖的新安全风险。

  2. 闭环 AI 自适应 — 对比器械均无实时、受试者特定自适应。处理方式：(a) 将 AI 定位为"辅助水平调整"（Class II 常见功能），而非"自主诊断/控制"（Class III）；(b) AI 仅提供辅助调整（缩放因子），而非生成新扭矩模式。

关键风险：BCI 新颖性

这是 Nexum One 最高单项注册风险。

问题： NMPA 对基于 BCI 的医疗器械审查经验有限。"神经再连接系统"监管框架尚未形成明确类别。审评员可能不熟悉 EEG 意图检测技术，倾向于要求完整临床试验。

乐观证据：
- BrainCo 智能仿生义肢（EEG 控制假手）在无完整临床试验下获批 Class II——建立了基于 EEG 的运动控制先例
- 纯 EEG 设备（博睿康、理邦）仅通过 CER 即为 Class II
- 外骨骼机械组件已有对比器械（UGO、DAAI）

风险降低策略：
1. 提交前咨询——最有效的单项风险缓解措施。向 CMDE 介绍拟议 CER 路径，在投入完整申报前获得反馈。
2. 聘请有 BCI 经验的 NMPA 顾问——曾参与 BrainCo 申报或类似产品。
3. 学术背书——获取中国 BCI 专家（如清华大学高上凯、天津大学明东）的推荐信，说明该技术文献充分，非实验性质。
4. 限制临床声称——将设备定位为"具有自适应辅助的步态训练工具"，而非"意图驱动的神经再连接系统"，以降低新颖性感知。
