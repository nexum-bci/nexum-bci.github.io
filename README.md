## 1. 执行摘要

**"让每个人重新学会行走。"**

Bones & Manifold 研发神经控制软体康复外骨骼——一款通过脑机接口与肌电感知运动意图、以柔性驱动辅助肢体运动、由神经力学模型驱动的可穿戴机器人。首款产品面向**全球中风康复市场**（9,400万患者，年新增1,200万），B2B医疗与B2C家庭双轨并行，中国NMPA、美国FDA、欧盟CE同步推进。

核心技术：**非侵入式EEG脑机接口 + sEMG肌电融合**实现0.2秒级运动意图识别（CMU Bin He实验室Nature Communications 2025验证）；**电缆驱动软体外骨骼**轻量级执行（Harvard Wyss方案，已通过Cybathlon竞赛验证）；**全身神经力学肌骨模型 + 个性化联邦强化学习**（pFedAC，arXiv:2605.14423）为每位患者生成持续优化的康复方案。

与竞品的根本差异：BrainCo做非侵入BCI但不做外骨骼；ReWalk/Ekso做刚性外骨骼但不用BCI也不用RL；Neuralink/Synchron做侵入BCI但距消费级10年。Bones & Manifold 是全球首个将**非侵入式BCI + 软体外骨骼 + 个性化RL康复**集成为单一消费级产品的公司。

## 4. 产品

### 4.1 产品定义：NeuroSuit — 神经控制软体康复外骨骼

**产品形态**：一件可穿戴软体外骨骼，像衣服一样穿在患者身上。包含三个模块：

| 模块 | 方案 | 说明 |
|------|------|------|
| **感知层** | EEG头带（8通道，干电极，<100g）+ sEMG手环（Meta Neural Band方案，16通道）+ 9轴IMU | EEG检测宏观运动意图（走/停/转身），sEMG解码精细力矩需求，IMU提供运动学反馈 |
| **执行层** | 电缆驱动软体外骨骼（Bowden cable + 伺服电机），髋关节+膝关节，总重<3kg | Harvard Wyss实验室方案，比刚性外骨骼轻70%，无关节压迫风险 |
| **智能层** | Manifold Omni：全身MSK模型 + RL康复策略 + pFedAC个性化 | 为每位患者建立个性化MSK模型，康复方案随恢复进度持续优化 |

**定价**：B2B版（医院/康复中心）¥150,000-250,000/台；B2C版（家庭康复）¥29,999 + ¥499/月订阅服务。

### 4.2 为什么软体而非刚性？

刚性外骨骼（Ekso, ReWalk, HAL）重12-25kg，穿脱需15-30分钟，关节对位不准可能造成二次损伤。软体外骨骼（Harvard Wyss方案）通过电缆模拟肌肉-肌腱的拉力，自然贴合人体关节活动范围——这是我们的MSK模型可以直接优化的控制策略。

### 4.3 为什么非侵入式BCI而非侵入式？

侵入式BCI（Neuralink, Synchron, NeuroX）信息带宽更高，但需要开颅手术、3-10年审批周期、单台成本$40K+。非侵入式EEG的信息带宽足以检测2-4个运动意图状态（走/停/左转/右转），CMU和Bielefeld大学已验证在线分类准确率>85%。对于康复外骨骼这一应用场景，4状态控制已可满足90%的日常康复需求。未来非侵入式BCI精度提升后，可OTA升级。

## 5. 核心技术

### 5.1 非侵入式EEG运动意图解码

基于CMU Bin He实验室方案（Nature Communications 2025）和Bielefeld大学免校准方案（Scientific Reports 2024）。8-16通道干电极EEG，卷积神经网络解码2-4种运动意图状态。关键突破：跨用户迁移学习消除个体校准需求——这在康复场景至关重要，患者无法承受30分钟的校准过程。

### 5.2 sEMG + IMU多模态融合

Meta CTRL-labs的Nature论文（2025年7月）验证了sEMG可检出单运动单元动作电位，16通道足以实现89-95%手势识别准确率。我们将其与9轴IMU融合，构建下肢运动意图的实时解码管线——这是目前学术界最成熟的非侵入式运动意图检测方案。

### 5.3 神经力学肌骨模型 + 个性化RL康复

这是Bones & Manifold的核心技术壁垒。每个中风患者的肌肉-肌腱-骨骼状态都不同（痉挛程度、肌力不对称、关节活动范围）。我们在全身MSK模型（416肌肉 + 206骨骼）上为每位患者构建个性化的数字孪生模型，通过RL在仿真中生成专属康复策略——竞品要么没有肌肉级建模（ReWalk/Ekso），要么没有RL个性化（HAL/ReWalk），要么两者都没有（所有中国竞品）。

### 5.4 pFedAC联邦学习

康复数据涉及医疗隐私，联邦学习是唯一可行的大规模部署方案。设备端上传模型更新、不上传原始数据——这在全球数据合规（GDPR, PIPL, HIPAA）层面具有结构性优势。患者越多，共享康复知识越强；单个患者使用越久，个性化方案越精准。
