# Kine
## 脑机接口人体增强机器人

**商业计划书** | 2026年6月


## 1. 产品

**Kine是一台脑机接口人体增强机器人。** 你戴上一副80克的EEG-Sense头带，穿上一件内置16通道sEMG织物传感器的NeuroSuit压缩裤。Kine读取你的脑电波和肌肉神经信号——在你迈步之前0.5-1.5秒，它已经知道你要动了。它在你需要的时候提供髋关节物理增强。你想站起来，它知道。你想加速，它知道。你走神了，它降低助力保护你。它不是穿在你身上的机器——它是你神经系统的外延。

**Kine不是外骨骼。** 外骨骼是预设助力曲线的省力工具——你动它就推，你不懂人，人也不懂机器。Kine是神经增强：读到你脑中的意图才辅助，读到你肌肉的疲劳就减力，像一个理解你的运动搭档。这不是技术路线的差异。这是产品定义的根本差异。

## 3. 技术

### 3.1 双模态神经接口：EEG + sEMG

运动意图不是单一信号。Kine同时读取两个层级的神经信号，覆盖从大脑意图到肌肉激活的完整通路。

**中枢神经——EEG-Sense头带**：8通道干电极（Emotiv/OpenBCI商用方案），80g，蓝牙连接。检测运动准备电位（MRCP）和感觉运动节律（SMR）。运动准备电位出现在你实际动作之前0.5-1.5秒——这是脑机接口的独特时间窗口。CMU Bin He实验室已发表非侵入EEG 2-4状态在线分类>85%精度（Nature Communications, 2025）。

**外周神经——NeuroSuit中的sEMG**：16通道织物电极（Nextiles技术，残奥会验证，可水洗50+次）。读出下肢肌肉的运动单元动作电位，<100ms延迟反馈精细力矩需求。Meta CTRL-labs已发表sEMG跨用户零校准方案（Nature, 2025）。

两种信号互补：EEG提供宏观意图预判（"要站起来了"），sEMG提供精细力矩控制（"用多大力"）。

### 3.2 全身神经力学数字孪生

416条肌肉、206块骨骼、Hill型柔顺肌腱——完整的人体运动系统数学模型。2448个仿真世界并行，日产53亿步态样本，零人工标注。在这个数字孪生上训练的任务无关运动策略（Manifold Omni）学习的是运动流形本身——不是上楼的规则、跑步的规则，而是人体运动作为一个连续空间的全部逻辑。单一神经网络覆盖9+种运动模态，无需模式切换。

### 3.3 pFedAC联邦持续学习

pFedAC（arXiv:2605.14423，CRO汪牧星第一作者）让每个用户的脑电和肌电模式被设备学习——但原始神经数据永不离开设备。只上传模型梯度，满足HIPAA/GDPR/PIPL合规。随设备数K线性加速。你的Kine每天比昨天更懂你。这是Kine的核心壁垒：不是单点技术，而是**四层耦合技术栈**——数字孪生→运动智能→联邦学习→双模态神经接口。极壳、程天、Ekso是硬件公司，同时补不齐这四层。

### 3.4 混合式机械架构

纯软体存在55%功率传输损失（Harvard Wyss实测）。纯刚性重12-25kg。Kine采用混合架构：碳纤维髋部锚定（刚性力传递起点，效率>70%）+ 织物包裹（柔性穿戴体验）+ Bowden电缆驱动（模拟肌肉-肌腱拉力路径）。总重<1.5kg，力传输效率>70%。


## 4. 产品形态

**Kine One（2027）**由三个模块组成：

**EEG-Sense**：8通道干电极头带，80g，佩戴像一副轻量耳机。持续监测运动准备电位——这是Kine最独特的传感器。它在你肌肉发力之前0.5-1.5秒就已经检测到你的运动意图。这个提前量让整个系统可以从容启动辅助力。

**NeuroSuit**：碳纤维髋部锚定+压缩织物+16通道sEMG织物电极。穿上如运动压缩裤。不启动时完全被动，不影响自然步态。AKE60电机通过Bowden电缆模拟肌肉-肌腱拉力路径，在髋关节双边提供辅助扭矩。设备端推理<0.5ms。离线工作。IP67防水。热插拔电池续航6小时。总重<1.5kg。

**Kine App**：蓝牙连接EEG-Sense和NeuroSuit。户外/都市/活力老人三种模式。实时神经数据可视化。每夜自主学习你的日间运动模式并优化明天助力策略。OTA固件持续升级。

**升级路线**：Gen 2（2028）+膝关节模块（$799），NeuroSuit扩展至膝部。Gen 3（2031）EEG-Sense升级16通道，+踝关节模块。Gen 4（2034）Kine Air：织物驱动，<1kg，全身可穿戴如发热衣。


## 5. 竞争

极壳$299，出货10万台——它是一部省力工具。Kine $1,499起，是一台脑机接口设备。用户支付的差价购买了三样极壳没有的能力：

1. **脑电预判**：EEG在你迈步前0.5-1.5秒已识别意图——辅助力在你发力前就准备好。极壳是"你动了我才推"（被动触发），Kine是"你要动了我已经准备好"（主动预判）
2. **双模态融合**：EEG（宏观意图）+ sEMG（精细力矩）同时工作。极壳只有IMU（惯性传感器），只能检测姿态，不能读取神经信号
3. **持续个性化**：pFedAC让Kine越用越懂你的神经模式。极壳的助力策略是出厂设置的，不会学习

类比：极壳是JBL蓝牙音箱，Kine是AirPods Pro——同一外形品类，完全不同的智能层级。Kine的竞争对手不是极壳。Kine定义的是**脑机接口可穿戴设备**这个新品类。目前这个品类没有第二名。


## 6. 为什么是现在

2025-2026年，四个独立趋势同时达到临界：

**神经科学临界**：非侵入EEG运动意图解码在线精度突破85%（CMU Bin He Lab, Nature Comms 2025）。sEMG织物传感器已商用（Nextiles, 2024残奥会部署）。Meta CTRL-labs Nature论文（2025年7月）验证了跨用户零校准sEMG的可行性——这是消费级神经接口规模化的前提。这三种技术的组合首次使$1,500以内的双模态神经接口成为现实。

**市场临界**：极壳5年从零到全球第一，$120M融资，证明消费者愿意为可穿戴运动辅助付费。程天¥2,599产品15秒售罄，证明老年群体同样意愿强烈。中国商务部2026年1-4月数据：智能助行外骨骼网零额同比增长785.5%。品类已从概念验证进入大众消费。

**政策临界**：中国十五五将脑机接口列为六大未来产业之一，首次写入政府工作报告。国家脑科学基金116亿人民币。工信部/民政部养老机器人试点（2025-2027）覆盖居家人群。江苏省2026年具身机器人消费补贴15%。

**物理AI临界**：2025年是人形机器人投资大年——宇树、Figure、1X、智元融资总额超$5B。但所有玩家都在做"让机器独立行走"。Kine做的是"让机器读懂人脑、增强人体"——这是物理AI更根本、更困难、也更大的命题。

**Phase 1（2027-28）：消费首发**：Kickstarter全球发布→DTC独立站→亚马逊/京东。B2C直营建立品牌和数据飞轮。每个用户的神经数据（保存在本地）持续优化其个性化模型，而联邦学习让所有用户的共享模型同步进化——这是一个随规模增长而增强的产品。
