# Nexum

**神经重连AI系统**

Nexum is an AI company. We build AI that reads brain signals, models the human body, and generates personalized motor assistance — creating a direct neural pathway from intention to action.

## Category

**人机共生 (Human-Machine Symbiosis)** — a neural system and a machine system fused into one body. Not an exoskeleton company. Not a BCI company. An AI company building the first human-machine symbiosis system.

## Product

**Nexum One** — neural reconnection AI system. Three modules:

- **EEG-Sense** — 8-channel dry-electrode headband (80g) for motor intention detection
- **NeuroSuit** — carbon-fiber anchored, fabric-wrapped, Bowden-cable-driven hip assistance (<1.5 kg)
- **Nexum App** — personalized AI strategy engine

Two versions: Clinical ($18,000, hospital) and Home ($3,999 + $499/yr, post-discharge).
