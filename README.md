## 1. 我们做什么

**Kine 构建物理AI——一种理解人体运动、增强人体运动的人造智能。**

人形机器人是物理AI在机器上的体现。Kine是物理AI在人体上的体现。前者让机器独立行走。后者让机器理解人的运动意图并与人体协同发力。后者更难——因为人体不是确定性系统。人体是过驱动的、非线性的、每块肌肉每根肌腱都在参与。

Kine的首个产品形态是可穿戴机器人。但Kine本质上是一家物理AI公司——我们构建人体运动的数字孪生，用强化学习训练它理解肌肉-肌腱层面的发力逻辑，然后把它嵌入一台你可以穿在身上的设备。

- 竞品（极壳/程天/Ekso）做的是**硬件**——更轻的电机、更便宜的结构
- Kine做的是**智能**——从肌肉-肌腱层面理解"为什么这一步该帮、下一步该等"
- 硬件会趋同。智能不会。

## 3. 技术

Kine的技术架构覆盖从材料到算法的完整链路。五个维度各自独立推进，最终集成为一台可穿戴的物理AI设备。

### 3.1 神经接口：双重感知

**sEMG——外周运动意图（Gen 1标配）**。16通道织物电极（Nextiles方案，残奥会验证，可水洗50+次）嵌入压缩裤，实时读取下肢肌肉的运动单元动作电位。结合9轴IMU（<10ms延迟）和足底压力传感器，在<100ms内解码运动意图。Meta CTRL-labs的Nature论文（2025年7月）验证了这一架构：16通道sEMG可实现89-95%手势识别精度，且支持跨用户零校准迁移学习。

**EEG——中枢认知状态（Gen 1选购，$299）**。4通道干电极头带（80g）不试图解码具体动作——那是当前非侵入EEG做不到的（在线精度仅57%）。它做一件当前技术可以做到且临床价值明确的事：检测运动专注度。CMU Bin He实验室（Nature Communications 2025）和Bielefeld大学（Scientific Reports 2024）均验证了EEG在2-4状态分类上>85%的在线精度。专注度下降→降低助力（安全）。专注度高→全助力（康复效果好——患者主动参与是康复效果的第一预测因子）。

两种神经接口读取不同层级的神经信号，互补而非竞争。这是目前全球首款同时集成外周与中枢神经接口的可穿戴设备。

### 3.2 混合式机械架构

纯软体外骨骼存在致命缺陷：55%的机械功率在软组织中损失（Harvard Wyss实验室，2016年实测数据）。纯刚性外骨骼重12-25kg，穿戴体验差。

Kine采用**混合式架构**：碳纤维髋部锚定（刚性力传递起点）+ 织物包裹（柔性穿戴体验）+ Bowden电缆驱动（模拟肌肉-肌腱拉力路径）。碳纤维锚定将力传递效率从纯软体的45%提升至>70%，织物包裹避免了刚性外骨骼的关节压迫和穿戴耻感。电缆驱动是当前最成熟的可穿戴驱动方案——Harvard Wyss的软体外骨骼（Science 2019）和ReWalk ReStore均采用此架构。

### 3.3 下一代驱动技术路线图

电缆驱动是Gen 1-2的实用选择。但Kine同时在跟踪三条下一代驱动技术路径，目标在Gen 3-4中逐步切换：

**气动人工肌肉（McKibben型）**：功率密度10.94 kW/kg（港中文ExoMuscle，Adv. Intelligent Systems 2023），是天然肌肉的39倍、电机的3-5倍。RT HealthTech已将其商业化至$150价位。当前瓶颈：寿命仅10,000次，需外部气源。预计2029-2031年解决。

**SMA织物肌肉（KIMM，韩国）**：25μm直径形状记忆合金丝编织成织物，10g织物可提起10-15kg。2025年10月完成自动化编织量产验证。<2kg全身辅助服装已在首尔国立大学医院测试，肌肉负担降低>40%。TRL 6-7，预计2028-2030年可穿戴化。

**HASEL致动器（Artimus Robotics，CU Boulder）**：功率密度>300 W/kg，响应<10ms，自修复，无声。已向美国海军交付。当前瓶颈：高压驱动微型化。预计2030+。

**生成式AI拓扑优化**：Caasenbrood等人（TU/e, 2020）已实现超弹性软体致动器的非线性拓扑优化——优化算法自动生成超越人类直觉的泪滴形波纹管几何。Chan等人（2024）将扩散模型（SDFusion）应用于3D软体气动致动器设计，可从文本描述生成新颖几何。Kine将在Gen 3致动器设计中引入生成式拓扑优化。

### 3.4 人体数字孪生与运动智能

416条肌肉，206块骨骼，Hill型柔顺肌腱。每条肌肉有独立的兴奋-收缩动力学（dA/dt = (S−A)/τ，舒张比激活慢4倍）。每根肌腱是非线性弹簧（Fse = Fpe + Fce，fp64牛顿迭代求解）。这不是刚体连杆近似——这是人体运动系统的数学模型。

2048个仿真世界并行，日产53亿步态样本，零人工标注。在这个数字孪生上训练的任务无关运动策略（Manifold Omni）不学习"上楼"或"跑步"的规则——它学习人体的运动流形本身。单一神经网络覆盖9+种运动模态，无需模式切换。

### 3.5 pFedAC联邦持续学习

pFedAC（arXiv:2605.14423，CRO汪牧星第一作者）是单时间尺度联邦Actor-Critic算法。设备只上传模型梯度，不上传原始数据——满足HIPAA/GDPR/PIPL合规。随设备数K线性加速。

## 4. 产品

### 4.1 产品定义

**Kine One**（2027）是一台混合式可穿戴物理AI设备——碳纤维髋部锚定（刚性力传递）+ 织物包裹（柔性穿戴）+ sEMG织物传感器（神经接口）。

穿脱像一条运动压缩裤。启动后，嵌在织物中的16通道sEMG传感器（Nextiles方案，残奥会验证）实时读取下肢肌肉的神经信号，结合IMU和足底压力传感器，在<100ms内判断你的运动意图——是要起身、加速、转弯还是停下。AKE60电机通过电缆（Bowden cable）模拟肌肉-肌腱拉力提供辅助，碳纤维髋部锚定确保力传递效率>70%（纯软体只有45%）。

**选购配件——EEG头带**（$299，4通道干电极，80g）：检测运动专注度而非解码具体动作。当用户注意力分散时自动降低助力（安全保护）；当用户高度专注时全助力。这对康复训练尤其关键——临床研究证明患者主动参与度是康复效果的第一预测因子。EEG提供的不是运动控制指令，而是**认知状态反馈**——这是非侵入EEG目前最可靠、最实用、最不忽悠的应用方式（CMU Bin He Lab, Nature Comms 2025；Bielefeld免校准方案, Sci Reports 2024）。

不启动时，Kine One是一条普通的压缩裤。启动后，它是你的第二层肌肉——能感知你的神经信号，理解你的专注状态，配合你的运动意图。

设备端嵌入式推理<0.5ms。离线工作，无需网络。IP67防水防尘。热插拔电池续航6小时。配套App切换三种模式（户外/都市/活力老人）。

### 4.2 产品哲学

Kine的产品哲学是**AI定义体验，硬件承载智能**。同一套硬件，通过软件升级不断提升智能化程度。你今天买的Kine One，一年后的OTA固件会比今天更懂你——它学会了你的步幅、你的节奏、你最喜欢的那条路。这是物理AI的iPhone时刻：硬件不换，智能持续进化。

## 6. 为什么是现在

2025-2026年，三个趋势同时达到临界点：

**技术临界**：sEMG织物传感器已商用（Nextiles残奥会验证，可水洗50+次），16通道即可实现>90%运动意图识别精度（Meta Nature 2025）。非侵入EEG运动意图解码精度在线>85%（CMU Nature Comms 2025），Gen 3集成。嵌入式AI推理<0.5ms。联邦学习框架（pFedAC）理论验证完成。

**市场临界**：极壳验证了消费外骨骼需求存在（5年全球第一、$1.2亿融资）。程天验证了老年用户愿意付费（15秒售罄）。中国商务部数据：智能助行外骨骼网零额同比增长785.5%。消费市场已经启动。

**政策临界**：中国十五五将脑机接口列为六大未来产业。具身智能列为优先产业。国家脑科学基金116亿。BCI首次写入政府工作报告。工信部/民政部养老机器人试点（2025-2027）。江苏省2026年具身机器人消费补贴15%。

**物理AI赛道**：人形机器人是2025年最热的投资赛道（宇树、Figure、1X、智元）。但所有人都在做"让机器独立行走"。没有人做"让机器理解人的运动并与人体协同"。后者是更大的市场，更深的壁垒，更根本的物理AI命题。Kine要做的是物理AI的另一个分支。
