# Kine
## 模块化可穿戴机器人

**商业计划书** | 2026年6月


## 1. 执行摘要

**Kine 让十亿人拥有第二层肌肉。**

全球60岁以上人口超过10亿，中国超过3亿，其中50%存在肌少症。每年因跌倒导致的医疗费用超过500亿美元。青少年脊柱侧弯检出率超过15%。户外登山人群中约60%因腿部不适放弃登顶。

与此同时，外骨骼市场在过去五年发生了根本性变化。极壳Hypershell从Kickstarter起步，五年内做到全球销量第一、B轮融资1.2亿美元、估值近4亿美元。程天科技消费级产品售价2,599元，上线15秒售罄。商务部数据显示，2026年1-4月智能助行外骨骼网零额同比增长785.5%。**消费级外骨骼已经从概念进入大众消费验证期。**

Kine的首款产品是**模块化髋关节可穿戴机器人**。它不像医疗器械——它像一副高端运动装备。3D打印碳纤维结构，AKE60电机驱动，重量1.5kg，IP67防水，热插拔电池。通过配套App切换户外/都市/活力老人三种软件模式。同一硬件平台可通过OTA升级实现不同助力策略。

核心技术：全身神经力学肌骨模型（416肌肉+206骨骼，Hill型柔顺肌腱）日产53亿步态样本用于RL训练；pFedAC个性化联邦强化学习（arXiv:2605.14423）实现单用户持续个性化。相比竞品的模式识别方案，Kine从肌肉-肌腱层面理解运动意图——每一步的助力都基于生理可解释的模型决策。

全球市场：**可穿戴外骨骼2025年$33.7B（Research and Markets），CAGR 32.05%**。其中消费级细分$200M，CAGR 19.6%；全球消费级外骨骼市场$1.69B，CAGR 19.6%。中国外骨骼市场2025年¥42B（中研普华），CAGR ~50%。前装市场（B2B医疗康复）$5.6B（MarketsandMarkets），CAGR 29.4%。总可寻址市场超过$40B。

**融资**：种子轮 ¥800万（原型+团队+认证准备）→ 天使轮 ¥2,500万（Kickstarter量产+首批交付）→ Pre-A ¥5,000万（规模化+海外+Gen 2）。全球Kickstarter首发，B2C消费+B2B医疗双轨并行。

## 3. 产品

### 3.1 Kine One — 模块化髋关节可穿戴机器人

**产品定义**：一台3D打印碳纤维结构的可穿戴髋关节机器人。AKE60电机驱动，重1.5kg。一部iPhone的重量分布在髋部两侧。IP67防水防尘。热插拔电池，一块续航6小时。通过蓝牙连接手机App，在户外/都市/活力老人三种模式间切换。

**这不是外骨骼——是可穿戴机器人**。外骨骼暗示医疗、笨重、需要帮助。Kine One是一件你在户外、通勤、日常中都愿意穿着的科技装备。

| 模块 | 方案 | 说明 |
|------|------|------|
| 关节 | 单髋关节，双侧辅助 | 最轻、最易消费化的形态 |
| 动力 | CubeMars AKE60系列，每侧1台 | 头部品牌通用方案，硬件不做无谓自研 |
| 结构 | 3D打印碳纤维框架 + 织物腰带 | 原型快速迭代，量产前开模 |
| 感知 | BNO055 9轴IMU ×2（髋部）+ 足底压力传感器 | 步态相位检测、意图预判 |
| 智能 | Teensy 4.1嵌入式控制器 + Manifold Omni | 设备端推理<0.5ms，无需云端 |
| App | 蓝牙连接，户外/都市/老人三种模式切换 | OTA固件升级，个性化调参 |

**定价路径**：工程样机 $2,500 → DFM小批 $1,500 → 量产综合均价 $1,499（户外套件$1,799，都市$1,099，老人$899）。

## 4. 核心技术

### 4.1 全身神经力学肌骨模型

416条肌肉+206块骨骼，Hill型柔顺肌腱——捕捉肌腱弹性储能与释放。基于MyoSuite+MuJoCo Warp，myowarp支持2048个仿真世界并行运行，日产53亿步态样本，零人工标注。这比传统OpenSim单线程仿真快两个数量级，比竞品的动捕实验室数据采集便宜数个数量级。

### 4.2 pFedAC个性化联邦强化学习

CRO汪牧星、杨朋昆、苏丽丽的论文（arXiv:2605.14423）提出单时间尺度联邦Actor-Critic算法。共享运动主干网络+本地个性化策略头。设备只上传模型更新不上传原始数据——满足GDPR/PIPL合规。随设备数K线性加速。

### 4.3 技术壁垒总结

| 层次 | 壁垒 | 竞品可复制性 |
|------|------|------------|
| 电机+结构 | AKE60+碳纤维3D打印 | 容易（供应链通用） |
| 传感器+嵌入式 | IMU+Teensy控制 | 容易 |
| **MSK模型仿真** | 416肌肉+206骨骼数字孪生 | **极难（3-5年积累）** |
| **RL个性化** | pFedAC联邦学习 | **极难（论文级算法+工程化）** |
| **数据飞轮** | 联邦学习+端侧隐私合规 | **极难（网络效应）** |

## 12. 愿景
