## 1. 愿景

**"让十亿人拥有第二层肌肉。"**

Bones & Manifold 是物理AI人体增强公司。我们研发可穿戴机器人——将脑机接口、柔性驱动、神经力学模型与强化学习运控集成为一件日常穿戴的智能衣物，帮助人类突破身体的生物学极限。

这不是外骨骼。这是**物理AI**在人体的延伸。

与宇树/Figure/特斯拉的人形机器人不同——他们建造独立行走的机器，我们建造**与人融合的机器**。物理AI的核心命题不只是"让机器像人一样运动"，更是"让机器理解人的运动意图并与之协同"。后者更难，也更大。


## 2. 执行摘要

全球**10亿人**（WHO 2025）需要运动辅助——中风幸存者、脊髓损伤者、截肢者、肌少症老人。这个市场远大于人形机器人——后者2025年仅$1.8B，预计十年后$38B。而全球康复机器人+BCI+老龄化辅助市场的总和已超过**$150B**，且渗透率不足2%。

Bones & Manifold 首款产品 **Eesaa One** 是一款面向全球市场的神经控制可穿戴康复机器人——通过非侵入式脑机接口（EEG头带）+ 肌电织物传感器（sEMG）+ IMU感知运动意图，以电缆驱动辅助肢体运动，由全身神经力学肌骨模型（416肌肉+206骨骼，Hill型柔顺肌腱）为每位用户生成个性化RL康复方案。

十五五规划将脑机接口列为"六大未来产业"之一，首次写入政府工作报告。具身智能同时被列为优先产业。**Bones & Manifold 恰好处于这两个国家战略的交汇点。** 全球运营从Day 1启动——中国NMPA、美国FDA、欧盟CE三条审批路径并行，临床数据跨国互认。


## 3. 产品：Eesaa（意识）

### 3.1 产品定义

**Eesaa One**（2027）是一件可日常穿戴的智能康复衣物。它不像医疗器械——它像高端运动装备。

| 组成部分 | 描述 |
|----------|------|
| **EEG头带** | 8通道干电极，<80g，像Marshall耳机一样戴在头上。检测运动意图（走/停/方向），<2秒窗口 |
| **sEMG织物袖套** | 16通道织物电极（Nextiles方案），像压缩衣一样穿在大腿/小腿上。解码力矩需求，<100ms延迟 |
| **IMU模组** | 6个9轴IMU嵌入织物，检测步态相位，<10ms延迟。最可靠的底层传感器 |
| **驱动腰带** | 腰部碳纤维腰带，内置伺服电机+电池+嵌入式控制器。像跑步腰包，<1.5kg |
| **电缆驱动** | Bowden cable从腰带延伸到脚踝，模拟肌肉-肌腱拉力。无刚性外骨骼框架——运动完全自由 |
| **智能核心** | Manifold Omni芯片：全身MSK数字孪生 + RL康复策略 + pFedAC联邦学习 |

**产品形态**：当不启动时，Eesaa One是一件普通的压缩运动衣。启动后，EEG+sEMG检测到运动意图，驱动腰带上的电机通过电缆提供辅助力——你感觉不到机器在"推"你，而是感觉自己的肌肉变强了。

**关键设计原则**：
- **无刚性框架**：电缆驱动+织物锚定。总重<3kg（含电池），是EksoNR（12kg）的四分之一
- **多模态融合**：IMU+EMG+EEG三者互补。IMU最快（10ms）但最浅（仅步态）；EEG最慢（2s）但最深（真实意图）
- **像iPhone一样设计**：可拆卸模块化，Gen1→Gen2→Gen3硬件可升级


### 3.2 为什么是物理AI

物理AI = 人工智能 + 物理世界交互。人形机器人是物理AI在机器上的体现，Eesaa是物理AI在人体上的体现。

| | 人形机器人（宇树/Figure/Tesla） | Eesaa（Bones & Manifold） |
|---|---|---|
| 交互对象 | 独立环境 | **与人体耦合** |
| 运动控制 | 确定性系统 | **非线性、不确定的人体** |
| 感知输入 | 视觉+IMU+力控 | **EEG+sEMG+IMU多模态意图** |
| AI模型 | 世界模型（外部环境） | **人体世界模型（内部生物力学）** |
| 安全问题 | 碰撞检测 | **关节损伤、组织压迫、寄生力** |
| 市场规模 | $1.8B→$38B（2035E） | **$150B+（已存在，渗透率<2%）** |

我们的MSK模型（416肌肉+206骨骼）本质上是一个**人体物理世界的数字孪生**——这和人形机器人的"世界模型"是同构的。联邦学习让每台设备的数据飞轮持续强化这个模型。

## 5. 核心技术壁垒

### 5.1 全身神经力学肌骨数字孪生

416肌肉+206骨骼+Hill型柔顺肌腱的全身模型。这不是学术玩具——它已经可以并行仿真2048个世界，日产53亿步态样本，零人工标注。每个用户的MSK数字孪生随使用持续更新。这是**人体物理世界的世界模型**。

### 5.2 多模态意图融合

IMU（<10ms）+ sEMG（<100ms）+ EEG（<2s）三模态互补。Transformer架构实现<50ms端到端延迟。CMU验证非侵入EEG 4状态>85%在线精度——对康复场景完全够用。

### 5.3 pFedAC联邦学习

康复数据是医疗隐私，不能上传云端。pFedAC（arXiv:2605.14423）实现：设备端训练，只上传模型更新——不上传患者数据。患者越多，共享知识越强；单患者使用越久，个性方案越准。满足HIPAA/GDPR/PIPL全套合规。

### 5.4 为什么竞品难以复制

| 壁垒层次 | 竞品可复制性 |
|----------|------------|
| 硬件（电机+织物） | 容易（供应链） |
| sEMG传感器 | 中等（Meta已开放方案） |

| **全身MSK模型仿真** | **极难（需要3-5年积累，竞品没有肌肉级建模）** |
| **RL个性化康复策略** | **极难（需要MSK模型+RL训练+临床数据闭环）** |
| **联邦学习数据飞轮** | **极难（需要论文级算法+分布式系统+合规架构）** |
| **NMPA/FDA审批** | **极难（3-5年时间壁垒）** |

## 7. 商业模式

**B2B首发 → B2C放量 → 数据平台变现**

| 阶段 | 模式 | 区域 | 定价 |
|------|------|------|------|
| **Phase 1** (2027-29) | B2B医院直销 | 中国+美国+欧洲并行 | $18,000-25,000/台 |
| **Phase 2** (2029-32) | B2B+B2C混合 | 医院+家庭康复 | B2C $3,999 + $499/年订阅 |
| **Phase 3** (2032-35) | 数据平台 | 全球 | $康复AI服务、保险数据、药物辅助 |

**全球运营从Day 1开始**：
- 中国：南京总部（制造+供应链），利用江苏BCI创新园政策
- 美国：波士顿办公室（临床+FDA），利用东北大学/MGH合作网络
- 欧洲：通过德国康复医院网络切入，CE认证（2029）

**为什么B2B而非直接B2C**：
ExoMuscle已验证$150消费外骨骼可行——但那只是简单助力，无BCI、无医疗、客单价太低。医疗器械级产品需要：①审批壁垒 = 竞争壁垒；②临床数据 = 信任货币；③医保支付 = 规模化前提。B2B建立这三者，B2C是水到渠成。
