## 3. 产品

### 3.1 NeuroSuit — 神经控制混合式康复外骨骼

**产品形态**：混合刚柔架构——关键力传递点采用碳纤维刚性锚定（减少55%功率损失），主体采用织物包裹减少压迫。电缆驱动（Bowden cable + 伺服电机）模拟肌肉-肌腱拉力——这是Harvard Wyss实验室验证的最成熟软体外骨骼方案。EEG头带 + sEMG织物电极手环 + 9轴IMU实现多模态运动意图感知。

| 模块 | 方案 | 说明 |
|------|------|------|
| **感知层** | EEG 8通道干电极头带（<100g）+ sEMG 16通道织物电极 + 9轴IMU | EEG提供宏观运动意图（走/停/方向），sEMG解码力矩需求，IMU反馈运动学 |
| **执行层** | 电缆驱动（Bowden cable）+ 混合锚定，髋+膝2关节，总重<3kg | 比刚性外骨骼轻70%，刚性锚定解决寄生力问题 |
| **智能层** | Manifold Omni：全身MSK模型（416肌肉+206骨骼）+ RL个性化康复策略 | 多模态融合延迟<50ms（Transformer架构），EEG以0.5-2s窗口做意图切换 |

**关键设计选择**：
- **电缆驱动而非气动**：UM-Arm气动方案续航仅12分钟，不适合临床使用。电缆驱动技术成熟（ReWalk/Ekso已验证）
- **混合刚柔而非纯软体**：纯软体存在55%功率传输损失（Harvard数据），混合锚定方案可将效率提升至70%+
- **织物传感器而非传统电极**：Nextiles已商用，可洗涤50+次，信号质量与凝胶电极相当——康复场景需要长期佩戴

## 4. 核心技术

### 4.1 多模态运动意图融合

我们采用IMU + EMG + EEG三层融合架构，而非押注单一BCI路径：
- **IMU**：<10ms延迟，检测步态相位和关节角度——最可靠的底层传感器
- **sEMG**：<100ms延迟，解码力矩需求和精细动作——Meta Neural Band已验证
- **EEG**：0.5-2s窗口，提供宏观意图切换（走/停/方向）——CMU验证>85%精度

三模态融合既保证了快速响应（IMU+EMG），又加入了脑机接口的"懂你"能力（EEG）。这是BIT MARX系统和Nature Comms 2024多模态融合路线验证过的可行架构。

### 4.2 神经力学肌骨模型 + 个性化RL康复

我们的核心技术壁垒。每个中风患者的肌肉-肌腱-骨骼状态都不同：痉挛程度、肌力不对称、关节活动范围、代偿模式。我们在全身MSK模型（416肌肉+206骨骼，Hill型柔顺肌腱）上为每位患者构建个性化数字孪生，通过RL在仿真中生成专属康复策略——康复师只需设定目标（如"改善步态对称性"），模型自动优化助力模式。

### 4.3 sEMG织物电极

使用Nextiles商用织物传感器方案（已验证残奥会应用），替代传统凝胶电极。可水洗50+次，信号质量与凝胶电极相当，解决了康复场景下长期佩戴的皮肤刺激和传感器脱落问题。

### 4.4 pFedAC联邦学习

康复数据属于医疗隐私。pFedAC（arXiv:2605.14423）实现设备端上传模型更新、不上传原始数据——满足GDPR/HIPAA/PIPL合规要求。患者数据不出医院即可参与全局康复知识共享。

| **B2C跟进**（2029-） | 线上直营+康复师推荐 | ¥29,999/台 + ¥499/月订阅 | 出院患者家庭康复 |

**B2B vs B2C决策逻辑**：
- B2B医疗：客单价高（¥15-25万）、审批壁垒=竞争壁垒、医保支付路径明确、临床数据可支撑B2C信任
- B2C家庭：需要B2B积累的临床数据和品牌信任、订阅模式提升LTV、中国市场77%出院患者无持续康复

**为什么不做纯B2C**：ExoMuscle已验证$150消费级外骨骼可行，但那是无BCI的简单助力设备，客单价太低无法支撑研发。医疗器械级产品必须走B2B审批路径来建立信任和定价权。
